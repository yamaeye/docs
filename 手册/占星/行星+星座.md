## 行星

```
没有一个行星能够让我们直接看到一个星座，现在它不再是红色的白羊和绿色的金牛，它变得复杂多了。土星可能会给红色镀上一层灰色，木星可能会给绿色镀上一层紫色。
```

十二星座像巨大的光轮一样笼罩着地球，各个行星在光环下运作着。它们以不断改变的速度来通过各个星座，有时停下来，有时向后退。行星通过星座时，像电线一样连接我们还不理解的力量，但是在传输的过程中，行星变换了光的性质。

媒介就意味着扭曲，每个行星都像过滤光的染色玻璃，在传送的同时也会给光上色。我们必须通过有色玻璃来看到光流，它们总是会将星座的光聚焦起来，然后将它对准地球，就像孩子用放大镜来点燃落叶一样。

光流在不断变动，而行星一直在星座的背景下移动，不断地传导不同颜色的光。不同行星中的光之彩虹每分钟都在改变，每一刻都不会重复。那无限变幻的，永远在更新的颜色，会在我们出生的那一刻凝固于内在。它是独一无二的，它是我们的身份，也是我们的目的，等待着通过一生时间来被揭示。

每颗行星都有一个独特的个性，每个都代表了人类意识的一个部门，每个星盘中都包含这十个部分，任何人都不会缺少任何一个部分。我们之间的区别只在于哪个部分是被强化的，以及我们如何表达它们。

如果我们可以敞开并有意愿地选择面对这些行星所代表的问题，它们将指引并教导我们如何让自己变得更加快乐。如果我们抗拒学习，它们就会如暴风雪直接刮到肌肤上一样严酷。如果我们选择去学习这门语言，就可以看到星盘背后的学习计划。

### 逆行

当一颗行星看起来反向运行时，我们就说它在逆行。地球是我们的快车，其它行星是跑得更慢的马。当我们还在后面时，眼睛看到的就是马在向前移动。但是当我们在轨道上超过它们时，就会看到它们好像停止，然后向后移动一会儿。

只有两颗行星比地球跑得更快：水星和金星，它们离太阳的距离更近。当两颗内行星围绕太阳运行时，我们会看到它们从太阳的一边荡到另一边，如此来回。就像是看小孩玩旋转木马，我们的视线紧跟他。当它们在太阳的前面通过时，是朝一个方向运行，当它们绕到太阳的后面去时，方向就改变了。

太阳不会逆行，它是这个系统的中心。我们总是围绕着它朝一个方向运行，这就造成了它也朝同一方向围绕着我们转的假象。月亮是太阳系中唯一一颗真正围绕地球运行的天体，我们是从一个固定的看台上看它，它的运行轨道不会改变，也不会有逆行现象。


行星是表达星座能量的通道，它们都是外向的。不过当一颗行星逆行时，它与通常的流向不同。它的极性反转了，不是以一种常规的外向模式向外流，它的能量的一部分被转向内在心识，远离世界，转向无意识。

当一个行星的力量不被转向世界时，它并不会蒸发。这颗行星作用的基本意义并没有改变，它还会继续，并且还是同样的有力。唯一改变的只是它的表现场所，那颗行星安静下来了。它安全地躲在心识里，跟外在那些会改变外向行星的社会程度隔绝，像一个野孩子一样成长。

逆行带来独立，它是完全自主的，因此可能也是危险的。如果你看到一颗逆行的行星，要像对待外国人一样对待它。他对文明充满戒备，在这里会感到不舒服。它感到尴尬和不安全，没有价值，虽然这些感受会披一件很大胆和防备的外衣。

逆行行星代表了性格中缺乏自信的部分，它感觉无所适从，很怕做错事。一颗逆行的行星常常会给那部分人格染上害羞和退缩的特色，它会很难表达自己。但与此同时，这个行星功能也会变得格外的有深度和敏感。

这颗行星是一个充满新鲜主意的金矿，但是它被埋得很深，将它挖出来需要耐心和策略。问题就是如何将这个野孩子从森林里哄出来，如果他带着自己的才能走出森林，那是因为他想选择如此做。

当一颗行星正要逆行或者准备恢复正向运行时，我们称之为静止。静止也不会改变一颗行星的核心意义，就像逆行一样，它只是调大了那个影响的音量。这个人心识的这个部分正在承受超高电压，它常常成为这个人性格中的一个顽固的主要特征。

### 火星-战斗

水星和金星跟太阳就像绑在一起一样，它们永远不会离太阳太远。但是火星这颗红色星球可不是这样，它是第一颗轨道在地球轨道之外的行星。火星能够去到自己想要去的地方，不受外界影响。

这颗夜空中的红色眼珠是自由的行星，独立，自我决断。火星象征着人类意志的力量，这颗行星给了我们去做自己喜欢做的事情的动力。去实现我们自己的生活，并且将任何阻挡我们这样去做的东西踩得粉碎。

这是热情的行星，如果火星被摧毁了，我们都会进入休眠。火星需要冒险，需要压力，他需要一个机会来达到自己的极限。面对恐惧就是我们需要知道的关于火星的全部，火星无法忍受懦夫。如果我们让恐惧来替自己做决定，那样一定会让这颗行星马上变得不爽。

```
火星是为老人的眼睛带来闪亮的生命火花的行星，是它让马拉松选手能够疾奔完成最后半英里，让一个小提琴演奏家能够在演奏帕格尼尼的急速乐章时能够有如此爆发力。在达豪的集中营，翁迪德尼的印第安人屠杀地，以及加尔各答，也正是火星支撑那里的人们保持了自己的意志。
他是一个凶猛而未驯化的斗士，当你在夜晚的街灯下被流氓挡住要钱的时候，它在这种时候是你的好伙伴。但是我们可能无法在一个安静的下午将他带到图书馆去——他可能会把那本厚重的韦氏辞典当铅球来掷，只是为了让他的热血沸腾起来。
```

当火星扮演一个重要角色时，那些人的个性是强烈而直接的。这是一种很危险的能量，当火星向坏的方向发展时，没有行星比它更加恐怖。它可能会变得无法形容的残忍、无法原谅的自私、无法理解的冷酷。

古人知道火星是战争之神，是果断和侵略的象征。它自大而易怒，当它移动时，他们都会害怕得发抖。冲突、不和、敌意、惊恐的确是火星的特质，但是将它们放在第一位是不得要领的。

只要将火星想要的东西喂养给他，就能保证火星成为生存本能与活下去的意志，而不是杀手本能。如果火星没有得到健康的释放，我们就会看到那个杀手出来了，易怒、爱争吵、为一些小事进行竞争。

```
代表自律和野心的摩羯座有一个秘密的梦想：单独一个人驾驶狗雪橇穿越拉布拉多，并写一本关于自己经历的书。这将是她的一个火星体验，因为它是多彩的，令人害怕的，需要很强烈的意志集中力。它也很摩羯，因为它需要一种持久有组织性的努力，它是单独的，也包含有一个事业的目的——她想写的书。
她最终必须去体验一些这种火星在摩羯座的冒险，可能是在拉布拉多，也可能不是。如果她如此确信自己必须去那里，但却没有依此行动，火星被压抑起来，她就会变得越来越不好相处。当然她身边的人不会忍受她太久，他们也会变得愤怒。于是她就收到了自己所发射出去的东西，她还是得面对这种不是她真正需要和想要的压力。
```

女神赫拉非常不满丈夫宙斯自行产下了雅典娜（据说她是从宙斯的头颅中诞生的），所以也不靠宙斯自己造出了阿瑞斯(Ars)。阿瑞斯是源自于愤怒和恶意，他是母亲狂怒的象征。战神阿瑞斯一向令人惧怕，而且因为易怒和难以控制的性格而被世人厌恶。阿瑞斯在战争中有三个随侍，分别是戴莫斯(Deimos，恐惧)、弗伯斯(Phobus，惊恐)和厄里斯(Eris，冲突)，这当然不是一个愉快的组合。

阿瑞斯在战场上的表现常出人意表，他非但没有所向无敌，反而很少获胜，甚至显得很愚蠢。当雅典娜轻松地用一块石头击中他时，他笨拙地坠落到地上，就像凡夫俗子般地尖叫大哭，甚至有两个凡人设法把阿瑞斯抓起来，将他囚禁在一个瓶子中长达十三个月之久（就像我们封锁或压抑自己的愤怒）。他不只在战争中表现得荒谬又可笑，在爱情的战场上表现也很糟糕。当他去诱奸阿佛洛狄忒时，很难堪地被她的丈夫赫准斯托斯(Hephaestus)发现，甚至还被赫淮斯托斯设下的金网困住。之后赫准斯托斯还邀请其他的天神前来，嘲笑伟大的阿瑞斯也有动弹不得的一天。

相比之下，罗马战神玛尔斯(Mars)就十分受人尊敬，他在万神庙的地位甚至高过伟大的天神朱庇特。罗马人非常崇拜玛尔斯，不仅因为他是战神，也因为他是春天之神，代表植物的生长和繁殖。他名字的拉丁文字根是mar或mas，意思是“照耀”和生殖力，他同时也被称为马尔斯·葛瑞戴维斯(Mars Gradivus)，源自于拉丁词根grandiri-，意思是变大和成长。他身旁的随侍是荷诺斯(Honos，荣耀)和沃特斯(Virtus，美德)。

希腊天神谴责阿瑞斯的粗俗和残忍，罗马天神却把玛尔斯奉为荣耀和美德的代表。这代表侵略性有两个基本面向：我们如果想要存活下来，而且成长茁壮，就必须宣告某些东西是不适用的，同时也无法推卸责任。火星既代表野蛮、盲目的愤怒、冲动和鲁莽，让自己显得很愚蠢，也代表一种健康的侵略性，亦即认识和掌控外在世界的正向冲动。

健康的侵略性是一种想要表达自己动物性的欲望，也是一种让种子发芽的力量。正向的火星可以让我们独立，使我们具备坚持立场的力量，并且能按照自己的意愿做出决定。健康的火星可以刺激我们去学习新技能，而这也是成功人生的基本要素（我们抓住一件事情，处理一个问题，然后解决一种困境）。

火星所落入的宫位代表着我们是能同时展现侵略性和魄力，还是只能发挥其一。侵略性和魄力的界线常常很模糊：小孩愤怒地反抗权威，这可能是一种侵略性，但也可能代表一种想要独立的欲望，而这种欲望对成长来说是非常重要而必要的。我们可以从火星的宫位看出必须在哪个生命领域采取攻势、冒险而又大胆无畏，同时还必须确立自己的主权、自由和独立性。我们在这个生命领域里，也可能遭遇战斗、过于激动、意外、过度竞争、暴力，以及对权力贪得无厌。

火星如果被“封锁”在一个宫位内，这个生命领域的经验就会让我们感到绝望、无助和沮丧，而我们也可能会在这个领域里挑衅惹怒他人。一个人应该在这个宫位代表的生命领域中展现勇气和活力，如此才能揭露自己独特的潜能，实现生命的计划和安排。当我们很沮丧或是心情低落时，往往会将注意力转移到宫头是白羊座的宫位，作为一种“重新开始”的手段。

#### 白羊座↑+↓天秤座

当一个人的上升点落入火象的白羊座时，就会用一种直接、坦率又充满活力的态度去面对生命，仿佛在说：“我在这里，注意我，让我们开始干活吧！”他会渴望坚决和果断，立即采取行动，并且“具备”创造或是掌握人生的力量。上升点白羊座的核心价值就是向内觉察一种创造性的潜能，而非站在原地等待着事情发生。在他们眼中，生命应该被视为一场探险、追寻或是挑战。如果星图中的其他相位显示了退缩或懦弱的天性，他们就必须更加努力，才能发展出白羊座的正面特质。

如果无法将上升点白羊座的特质展现出来，内心的挫折感经过日积月累，最后就会周期性地生病，爆发激动的情绪，不时地失控或发怒，或是表现出其他自我毁灭的行为。如果星图中其他部分的火象特质已经十分明显，上升点在白羊座的人就会随心所欲地展现魄力和自信，但也有可能变得太过强势或过度自我中心，最后可能落得弄巧成拙，一败涂地。

白羊座对面的天秤座会落入下降点，这就是要提醒一个人多少得考虑他人的立场，借此来平衡过度的自我意识。但如果上升点在白羊座的人必须在两极之中做出选择，较为明智的做法可能是与其为了维持和平而有所保留，或是为了他人的需求而改变自己，不如因为自作主张或是大胆冒险而犯错。他们必须先找到自己的力量，大胆地活出自己之后，才能学会管理、调整并且缓和白羊座的本质。到最后，才能将天秤座的优雅、深谋远虑和考虑他人的观点，融入到自己的本质中。

上升点在白羊座的人可能生来就五官鲜明，看起来朝气蓬勃、充满热情，目光总是直视对方。他们的行动通常迅速又冲动，仿佛总是随时准备好一头栽入新的事物中。当他们与其他人沟通时，总是隐约地（或是不怎么含蓄地)传达出一个信息：“别挡住我的路。”

```

```

###### 形象

最初一切都不存在，然后出现了什么。生命将自己从空无之中投射出来，在渺渺虚空之中创造出自己的地盘。如同火山爆发，白热的恒星爆发成星云碎片。这个过程是无所畏惧的、爆发性且猛烈的。白羊座是生命力，存在的意志。低下头露出羊角，蓄势待发，凶猛而不屈服。这撞击是否会撞裂头颅？他不在乎，任何事情都无法让他害怕，无法让意志改变方向。要么胜利，要么在追寻胜利的过程中自我毁灭，他只会进入这两条路中的其中一个。

```
一只疯狗将你逼到了悬崖边上，它犬牙毕露，口中含着白沫。它逼近你，嘶吼着，眼中泛红，闪着凶光。你振作起来，没有言语、风度和教养的假面具可言，只剩下动物性的生存愤怒。你手中握着猎刀，大叫着直取它的喉部。你找到了自己的白羊座。
```

###### 主题

白羊座的大胆行为可能是任何冒险，但它的真正含义代表存在的勇气，教导我们通过意志战胜任何威胁、障碍和疑虑的能力。白羊是内在去做它想做事情的部分，选择并行动，其它的一切都无关紧要。唯一吸引白羊注意力的就是去塑造生活的意志力量，终点是欲望和行动的完美结合，也就是自由。不会迷失在感受中，对决定背后的心理基础也不会迷恋。

自私是很微妙的艺术，并非不敏感，也不是狭小或操纵。而是去说：“这是我的生活，我有权利找寻任何我需要拥有的体验。任何东西都不能挡在我和我的成长之间。没有任何一个人，任何一种境遇可以，甚至包括我的恐惧。 ”

```
我们被一根绳子悬在一面光秃秃的山崖的半中间，如果那根绳子没有问题，也系得很牢靠，而我们又不犯错误的话，那天晚上就很可能会活下来。在这种情况下，如果不害怕，那肯定是脑子有问题。但是攀岩者学会了控制恐惧，在他所从事的运动的情绪压力中保持高效地工作。
```
压力与勇气是很相配的一对，勇气必须被吓出来。令人害怕的境遇并不会每次都让我们变得更加勇敢，但没有它们，我们永远无法通过其它途径变得更加勇敢。白羊座找寻勇气，也吸引压力，危机像阴影一样跟随白羊座。完全去做自己喜欢的事情，这种勇敢很难实现，它所制造的危机都很激烈。如何去面对压力是无法被预测的，是否去征服它也是自己的选择。

爬一座山需要胆量，这是静态的论断，此过程中的动态变化可以换另一种方式表达，爬山会唤起一个人的勇气。很多时候我们都是在一个更安全，不那么危险的地方去练习这种存在主义的勇气。白羊座的经典策略是故意选择一个危机，冒险可以发展出勇气，自然的情绪被克服了，意识的结构也被改变了，这种转变是成长的本质。

每个人都有自己的山，不管一个白羊是否愿意去迎接挑战，生活都会给他一座看起来再普通不过的山。危险的体验就像寒冷长夜之后的朝阳一般吸引着白羊，这个隐藏着的渴望被包裹在恐惧的面纱之下。白羊的艺术就是去揭示这个需要，不管多么令人恐惧，不惜一切代价去满足它。与恐惧同在，并清晰而坚决地行动。


```
对一个人来说，它可能是去学习游泳；对另一个人来说，可能是去质问他专横的上司；对第三个人来说，可能是去戒烟。朋友可能会离开，权威人士可能会镇压你。他可能会坚决而创造地面对，也可能会像兔子看到推土机一样逃掉。
```

###### 优势

```
自然让白羊为它的山做好了准备。这种好斗有时显现在明显的物理层面上，就像攀岩者对一座山一样。但通常不是那么明显，可能是一种好辩，可能会看到一个保护妇女权益或环境的斗士。也可能会看到跟自己内在的魔鬼不断交战的人——一个戒酒者，一个跟自己的噩梦交战的退伍越战老兵。
```
白羊的灵魂非常强烈而直接，坦诚、活跃、热情、独立，还有对权威的反感，生来就具有好战性、活力和生存的本能。不管他在这世界上穿着一件怎样的外衣，在那下面都跳动着一颗战士的心。有可能通过一场危机才能将它逼出来，将最温顺的白羊座人放进一场危机中，也能够观察到一种冷酷的、清晰的效率。在对压力的期待之下，膝盖可能会颤抖，过后可能会晕倒，但真正的白羊特质会发出光芒。能够在星盘上白羊出现的任何地方找到好斗的倾向，会看到有一股战火烧向何处，目标只是迟早的问题。

###### 劣势

一个战士在和平时期该做什么呢？他无法在外在没有任何抵抗他的威胁时保持一个自我形象。军队需要敌人，战士需要战斗的热情完成来此的使命，但如果他不去面对进化的危机，那团火还在那里，它不会熄灭。相反，它会去攻击一个外在目标，而没有被用在真正的问题上。如果他们没有被一个合理的敌人挑战，他们就会找一个不合理的敌人，可能是反对他们本来要保护的国家。

白羊座一定会战斗，但出于无聊、易怒和沮丧，战士也可能打一场错误的战斗。无止境、无目的、无意义的冲突争论，常常伴随跟内容不相符的强烈情绪，所有的猛烈、意志和勇气被毫无目的地消耗了。进化的、助推的火箭燃料像国庆焰火一样冲向天空，冒着硫磺和地狱之火，没有任何赢家。选择了这条路的白羊因此受伤，充满了饥渴和挫败感，朋友们被赶走了，婚姻失败了，事业也因为小问题而没有了。

幸运地是，合理的敌人多的是，一次又一次地，白羊会遇到路障。可能一个想要夺走自由的权威者，也可能是各种困境的组合，有时候是内在的软弱。通常路障以双方误解的形式呈现，而这种误解往往源于白羊自我的强烈度，他的问题总是同样的，这种特质常常让别人进入防卫状态。路障必须被面对，如果对白羊来说有一种最重要的罪，它一定是胆怯逃跑。

白羊是一块压力的磁铁，选择只在于压力会是哪种形式，可能是一个有意义的个人成长、接受挑战、还有征服高山。这种战斗的吸引力很模糊，它们能帮助我们成长进化，但我们可能也很累。战斗必须是自愿接受的，白羊也可以自由选择最小阻力的那条路。

### 金星-魅力

```
太阳在一片雪景之中慢慢西落，远处大树的枝条仿佛艺术品一般。空气中的湿气都被冻结了，天空清澈如镜。白雪反射的强光减弱形成影子，锐利的线条和边缘开始显现。灰色和黑色成了主要的景色，天空的蓝色也慢慢变暗，从天蓝变成靛蓝，从靛蓝变成初夜的紫色。突然之间金星出现了，发着光，如此华丽闪亮，美得让人屏住呼吸。
```

如果太阳是国王，月亮是王后，那么天庭中的公主一定是金星。金星曾被数千年崇拜，被称为夜晚之星。跟日月的光相比，她的光是暗淡的，但她比所有其它空中的天体都要亮。金星是天庭中最精美的珠宝，像一颗发光的珍珠，如闪电一般雪亮，但却带着一种更加柔和的气质。

均衡和谐、平衡是金星振动中的关键，她在星盘上的位置告诉我们可以让自己变得平静的体验。内在宁静是她的天赋，金星是来自天国的镇静剂，能够协调一个人内在的所有矛盾因素。在旧的传统中，她是和平之神——思想的宁静以及国家之间的和平。

我们每个人生下来都有一种渴望，这种渴望的解药并不容易被知道，更不用说被找到。即使是短暂地满足那个渴望，都是金星的艺术。她的方法多样，但目的却永远相同：帮助我们找到一刻的宁静。

这宁静可能会来自爱人的臂膀或是一首动人的曲调，或者在一座深山中突然超越自我的灵光一现。不管它在哪里，金星都会将我们带到那，只要我们去倾听。我们能够听从她的建议或者忽略它，这个选择永远都在自己。

金星是美神，为我们带来对美的感知和艺术的倾向，带领我们在颜色、形状、声音中创造出和谐，或者对已经存在的和谐做出反应。受它的影响很大的人是那些让我们所有人快乐的画家、音乐家、演员、还有诗人。

当金星在一个人的星盘中比较重要时，他们越是接受自己是一个“艺术家”的形象，主动地发展自己的创造能力，他们就越会感觉到跟自己以及世界和谐。如果他们拒绝这一点，他们会找不到自己的深层精神停泊处。

```
这个人总是会在自己的客厅墙上挂有油画，或者有一套有趣的音乐收藏。如果问他们有没有个人的创作作品，他们可能会说自己笨手笨脚——但是在阁楼的某个角落里，一定藏了一堆诗作或者一把老吉他。如果你能够诱骗他们弹上一首曲子，你会震惊得目瞪口呆：感觉自己第一次看到他真实的样子。
```

金星是最靠近地球的行星。另一个邻居火星，离我们的距离永远是金星的两倍以上，其它的行星就要远更多了。在质量和直径两方面，金星就像是地球的双胞胎。如果不是充满硫酸的大气与高温，金星更像是地球的热带版本。

金星离我们很近，她外表跟我们很像，是天空中最亮的行星。她除了是爱神，还能是什么呢？我们都生来感觉自己只有一半，生命的大部分时间花在了找寻另一半上，一旦找到之后，又花在如何让它和谐上。

跟自己爱的人的互动是如此至关重要的主题，如果没有外在的回馈，我们无法在关于爱的学习上有什么进展。我们必须有伴侣，金星知道这一点，所以她为我们的关系做广告。为我们带来肉体的美，一种无法抗拒的动物般的优雅。她也会带来人格的吸引力，镇定、礼貌、高贵的言谈举止。

一个金星型的人永远不会强迫，而是像波浪一样冲刷你，在我们发现之前，我们已经被淹没了。但优雅也可能退化成油滑，魅力也可能变成操纵力。她可能会成为一个花花公子或者蛇蝎美人，也可能变成一个充满了疲惫、消极、奢侈逸乐的肉欲之人。

就像水星一样，金星比地球更接近太阳，这也让她跟自我绑在一起。她可能会跌落成为虚荣和占有，将人类的爱变成一系列肥皂剧式的场景。最坏的情况是，我们的金星成了一个过气的电影女王，躺在她的长沙发上，周围满是发黄的宣传照，很肥，但是依旧虚荣的吃着巧克力。

对几乎所有人来说，关系都是生命中至关重要的事情。金星象征我们建立关系的能力，她所代表的问题也是我们所面对的最强迫性和最令人疑惑的问题。金星在星盘上的星座告诉了我们这个人的需要，他或者她会觉得什么是吸引人的。

```
金星在双子座会很需要来自伴侣的心智启发——新鲜的、有启发的交谈，以及想要尝试新鲜体验的意愿。还有一种对各种不同的伴侣的需要——如果不是爱人，那么就是朋友。当金星在巨蟹座时，状况就完全不同了。在这里它的需要就安静得多，更加情感化。一夫一妻来得容易得多，并且更容易被接受。
```

金星象征着所有人对于结盟和关系的渴望，阿佛洛狄忒就像月亮一样，代表一种平衡、协调、融合与真爱的阴性法则。我们可以根据金星落入的宫位，找出在哪个生命领域中，可以自然地感受到和平、宁静、幸福和满足，同时也会在这一个领域里产生欣赏、尊重、爱人与被爱的能力。这是一个最令人愉快的领域，也是最能散开心胸、被他人取悦的领域。我们可以在这里展现最佳的风格、品味以及对他人的体贴。

这位女神也有一些令人不太愉快的本质，她不能忍受他人无法达到自己的理想和标准。基于阿佛洛狄忒对于完美及和谐的高度期许，金星落入的宫位也代表我们无法达到理想时，就感到失望和幻灭的生命领域。然而，就是因为受到这种不满意的刺激，所以金星落入的宫位也代表我们会在这个生命领域里被迫采取行动，让世界（或者是我们自己）变得更公正、更公平，也更美好。

当天神帕里斯(Paris)想要从阿佛洛狄忒、雅典娜和赫拉之间选出最美丽的女神时，阿佛洛狄忒毫不害臊地脱下衣服，想要借此左右帕里斯的看法。阿佛洛狄忒为了确保帕里斯不会选择其他两位女神，进一步地去收买帕里斯，将迷人的海伦送给他当妻子，完全不顾海伦才刚刚成婚。对于她而言，赢得选美才是更为重要的事情。这个安排掀起了特洛伊战争，让数以千计的民众陷入悲惨和动乱之中。

这位代表爱和美的女神，有时也会让人们的生活变得一团糟。金星落入的宫位也代表某种程度的痛苦、冲突和苦难，因为当我们太过于偏向一个方向时，就必须经历苦痛，才能重新找到更好的协调或平衡。阿佛洛狄忒偶尔也会去调解一下不平衡的事情，她曾经鼓励儿子厄洛斯(Eros)用他最有名的弓箭把普鲁托扯下来，让年轻的珀耳塞福涅重获自由。在她看来，珀耳塞福涅实在太过于天真无知，不懂得为自己着想，所以她必须出手介入，没想到却因此而严重地扰乱了珀耳塞福涅的生活。

#### 金牛座↑+↓天蝎座

若是上升点落在土象的金牛座上面，这个人就必须用比较缓慢又从容的态度去面对生命。上升点金牛座必须规划、组织，并且有条不素地朝目标迈进，而不是冲动地一头栽入。与上升点在白羊座的人相比，上升点金牛座的稳定性意味着此人必须花更长的时间，维持或停留在每个阶段的经验里，但他们也可能会变得太过懒惰或是好逸恶劳。

当他们确知某些纯粹出自兴趣、归属感或安全感的事物的确具有目的和实用性时，就很可能会执着于这些事物不放。上升点在金牛座的人必须学会什么时候该坚持，什么时候该放手。他们必须在实际的物质生命领域里感到舒适，努力地耕耘以获得具体的成果，同时必须培养一种对于身体和生理需求的觉察能力。当他们过度地被本能所驱策或掌控，并不是一种最好的状态，但与基本天性完全切割也不是一件好事。

下降点是天蝎座意味着他们在关系的领域里时常会挑起强烈又极端的情绪（天蝎座），忌妒和占有欲时常是人际关系问题的本源，必须去面对和检视自己情感本质中的破坏性面向，并且学着去掌控它。落入第七宫宫头的天蝎座，会强迫自己去观察内心，探究其中隐藏的动机和潜在的因素。唯有如此，他们才能改变自己运用权力的方式。下降点天蝎座不允许一个人只看事物的表面价值，关系的问题会刺激他们从一成不变的生活中觉醒，定期地清除或净化累积在内在的东西，改变老旧和陈腐的习惯模式。

上升点在金牛座的人会有两种不同的体型。比较常见的是结实、稳重且壮硕：健康、有着如“干草卷”般的“乡村”模样，或是有如福斯塔夫（莎士比亚作品《亨利五世》中一个耽于享乐、丰满多肉的人物)般的外表，通常他们都会有金牛座的粗壮脖颈。另一种上升点在金牛座的人外型较为精致，有如雕刻般的容貌，展现了金牛座的守护星座金星的美感。

```
女演员费雯·丽（电影《乱世佳人》中郝思嘉的粉演者)就是后一类型的代表，她的上升点落入金牛座，而金牛座的守护行星金星又落入第十宫。英国抒情诗人珀西·比希·雪莱常被人形容是纤细又柔弱的，他的上升点落在金牛座，守护行星金星落入展现创造力的第五宫。美国诗人沃尔特·惠特曼的作品充满了对大自然和生命的丰富赏析，他则是金星与落在金牛座的上升点合相。
```

###### 形象

被白羊座滚烫的火炉烧得筋疲力尽之后，意识开始寻找清凉的池水、古老石头的安慰。白羊的战争已经取胜了，金牛获得了宁静，用寂静取代激情，用交响乐取代高谈阔论。绿叶之间传出疗愈的鸟鸣声，精神不再被战斗之火点燃，它将饱经风霜的手伸进土地中，感觉种子、泥土、地球。

金牛座的符号是一只公牛，不是对着红布发怒的野兽一般的凶猛斗牛，而是一只安静的公牛。阳春四月独自站在橡树下，静静地看着绿草、太阳、小牛和大地。没有任何事情让他害怕，白羊的恐惧姿态已经走了。金牛对自己的世界有如此的控制度，他是如此超越恐惧之上，以至于无惧本身对他都没有意义了。


###### 主题

五月初的悬崖顶端，暖风缓缓撕扯着天上懒懒堆积的云，山茱萸绽放着，鹰在天空中毫不费力地盘旋着。你独自坐着，在温暖的空气中，感受屁股底下被太阳烤得暖暖的石头，春天的触感如此让人舒心。在你面前山谷延展开去，开满了鲜花，农民在肥沃的田地里忙碌着，牛在吃草。春天的绿色一段段地蔓延过犁沟，一直伸向远处的森林。

你在那儿坐了一个小时，两个小时，然后又是一个小时。没有思考，只是感受。没有什么问题在困扰你，你对生命的本质并不关心。在那个时刻，你只是存在，语言不再必要。在那广阔的内在空间里，你的感受是永恒，安详，宁静，找到那份安详并保持它，就是金牛的任务。肥沃的土地教给我们，如何变得无限复杂但是依旧简单，如何变得不可探测的深刻，而不需谈论它。


###### 课程

纯正的金牛总是会选择乡村的舒适和安静，在树林中的一次散步、在泉水边静静地待一个小时、夜晚走过沙漠的灌木丛，这些都比“说话疗愈的方式”给金牛带来更多的宁静。地球母亲是金牛主要的老师，她抚慰精神，教导简单和平静。并不是所有的金牛都那么爱大自然，但坐在山顶是一个基本的进化策略。其它力量也可能会将金牛部分很强的人带到都市里，在那个环境里维持宁静是更难的，定期离开城市可以帮助保持跟地球的连接。

金牛是所有星座中最沉默寡言的，会直觉地反对说太多话，并发现说话是一件很难的事。沉默孕育简单，简单孕育宁静。金牛的真正目标是内在的宁静，他们的本质跟语言是相悖的，无法用语言来翻译。谁管它呢，完美的此刻就是生命的本性，去谈论它只会让面纱再次落下。

当我们被一首旋律的音符所催眠，完全地沉浸在倾听之中时，或者当我们演奏音乐的时候，沉默的效用更明显，头脑中一直在持续的说话声沉默了，停止了自言自语。金牛想脱离头脑脱缰的想象所产生的压力，而扎根在物质世界。必须要去碰触，通过感官、皮肤、指尖来找到世界，永远无法只是通过头脑来找到它。他要通过身体自我疗愈，他不会超越肉体，而是纵情于其中、发光于其中、欢庆于其中。

```
有一个装满植物的房间，一只猫或者狗。可能是周末登山，或者在乡村的小旅店里待几天。手伸进泥土中，手指轻抚一支高品质的老旧小提琴，用黏土、油漆来创造，或者即使是打扫房间或切蔬菜。感觉大地在脚下，听到音乐在耳朵里，感觉伴侣的身体贴在身体上。向后躺在自己的皮椅上，感觉到一个完美的苹果在嘴中脆脆的口感，看着自己的孩子，感觉到房子的坚实，身体的高效。
在淋浴中歌唱，或来段即兴的口琴演奏，在一屋子人面前演奏一段肖邦练习曲当然也有同样效果，不过这只有在我们将所有的公开演出时的自满、虚假、还有紧张都置于演奏音符的享受之下时，才能发生。
```

###### 优势

对戏剧性的反感，对复杂的怀疑。本能地，金牛会寻找那些能够让他找到自己的简单宁静的境遇。稳定的工作，可靠的关系网。耐心地，绝对实际地，金牛编织着一个茧。在这个茧里，他的工作有条不紊而不可阻挡地前进，就像石钟乳在缓慢成长一样。金牛此刻就活得很好，在无法言说的内在深处，一个无法探测的沉默部分，他能感觉到一些其它星座无法感觉的东西，他感觉到敬畏。


###### 劣势

古老的山与大橡树等不朽的庞然大物只是象征，金牛找寻的是内在安全感所孕育出的永恒宁静。但他可能被外在世界的映射所迷惑，构建一种产生致命吸引力的“愚人的黄金”，只是去累计物质安全感，金钱、财产、地产、退休计划……金牛迟早都要面对十字路口，生命是一个防卫性的还是一个进攻性的行为？我来这里是为了成长还是为了安全？

如果屈服诱惑选择一条可预测的、物质安全的路，这条路可能是诚实和道德的，对所有人都无害，但麻木的、会杀死精神的稳定会让金牛无聊至死，失败的金牛可能会失去一切：“我昨天、今天和明天都是一样的。”另一条路则会带来成长和改变，在此过程中，自我形象及与之相应的态度也必须改变，不确定，不安全，但是令人心动，这正是生命的基本要素。

安全感也可能在头脑层面产生阻塞，金牛有种意见坚定的顽固，可能造成体验的狭隘和没有想象力的反应，让人无法进行那种存在主义式的、凭着信念的纵身一跃。虽然很多时候他表面没有很多意见，可不要将安静之下的钢铁意志错认为软弱无力，它带来了可以完成任何决心要做的事情的能力，但也会带来不变通。

#### 天秤座↑+↓白羊座

上升点在白羊座的人会表现出顽强、果决和自作主张的行为，但上升点落在天秤座的人则截然不同，他们会要求必须客观公平地评估每一种情况，在经过深思熟虑后小心翼翼地采取行动，其他人的需求和想法都必须纳入考虑。上升点天秤座的基调就是反射性的判断，意即在平衡中衡量各种可能性，然后选择最适当的生存方式或行为。但察觉他人的观点或是从各种角度去考虑事情的倾向，有时反而会成为一块绊脚石，阻碍他们采取行动的能力。因此上升点在天秤座的人大多以犹豫不决、优柔寡断和观望的态度闻名。

我们如果能有一套基础的价值体系，就比较容易做决定，因此上升点在天秤座的人肩负的最沉重功课就是建立一套价值、标准和理想。让其他人替自己做决定是一件很容易的事，但即使天秤座是根据自认的真实和公正去采取行动，就一定保证会有完美漂亮的结果吗？他们应该学会如何做决定，同时为自己的决定负责。

除了必须在生命的“阳性面”和“阴性面”之间取得平衡之外，也必须在神秘的渴望和实际的想法之间、理智和心灵之间、直觉和逻辑之间，还有自我的需求和他人的需求之间，找到一个平衡点。上升点在天秤座的人追求完美的关系、理想的哲学观，以及任何令人感到和谐和愉悦的事物。他们通常会对艺术或类似政治和数学的抽象理论感兴趣，因为这些东西能够展现一种对称而完整的概念。

很多上升点在天秤座的人可能有过于崇高的理想或信仰，却与现实的艰困面完全脱节，当一件事没有达到渴望或预期的标准时，上升点在天秤座的人可能会变得十分挑剔。他们往往非常强势地说：“为了和谐，你最好看看我的观点和方法。”上升点在天秤座的人身材比例非常匀称，体态迷人，但很容易因为懒惰或是过度沉阔而发胖。风象星座落在上升点时，外表通常都比较精致而非粗线条。

伴侣关系是非常重要的，而这也是他们成长的必经之路。在与伴侣有关的生命领域中遇到下降点白羊座，可以引发出天秤座的本质。如果伴侣是一位非常自我中心又果断的人（白羊座类型），上升点在天秤座的人就必须调整或让步。然而，当对方变得太不公平、太过于强势或是要求太多时，上升点在天秤座的人就必须学着为自己挺身而出，要求天秤座所注重的公平与和谐。别忘了，对立的事物总有办法让事情逆转。当上升点在天秤座的人不喜欢你的演出方式时，他很快就会让你知道这一点。

```
根据阿道夫·希特勒儿时洗礼仪式的记录来看，他的上升点就落在天秤座，凸显了天秤座会为了自己的哲学或政治理念而容许残暴，甚至毁灭一切不符合自认为和谐或完美的事物。上升点天秤座正面特质的例子，在诗人卡尔·桑德伯格(Carl Sandburg)的出生星图中，金星双鱼座落入代表创作表现的第五宫，温斯顿·丘吉尔的金星则落入负责沟通的第三宫。
```

#### 天秤座

```
只有在寂静中才有言语，于黑暗中才有光芒，在死亡里才有生命，雄鹰在空无一物的天际才闪闪发亮。 
——厄休拉 ·勒奎恩
```

###### 形象

天秤不是现代的产品，出自巴比伦和古埃及的市场，没有弹簧或数字显示屏，只有简单的杠杆。直接将一盎司的铅放在托盘上，在另一个托盘上倒金沙，当它们彼此平衡时，就有了一盎司的金子。铅遇到金，生遇到死，爱遭遇恐惧。没有光是不带影子的，没有阴影是没有光的，天秤代表和谐、均衡、两极的和解。

###### 主题

```
雨夜，在佐治亚州公交车站，车引擎烧坏了，抛锚在三英里之外的地方。左边电视上的付费广告不停让人用更多除臭剂，令人疲惫，右边手提式大功率录音机放着刺耳的三岁小孩听的迪斯科舞曲。扩音器里传出通知，公车还要晚点两个小时。我们在乎吗？不，这就是生活。没有理由苦恼。
```

天秤教导安详，代表心识有意识地控制神经系统的能力，内在不会被任何东西所扰动、激怒、震惊的部分，跟混乱签署了永久的和平协议。天秤座的目标是意识的转化，目的地就是平静下来，要获得牢不可破的内在和谐，没有什么能够动摇或打破均衡。旋转木马的中心，站立着有意识然而不为所动的天秤。伤痛来了，伤痛走了，喜悦和悲伤起伏，就像黑夜与白天交替。

###### 课程

其它星座有一种内置的平衡，他们的任务跟平静下来无关，不管生活变得如何不和谐，都会继续将注意力放在手中的事情上。而天秤对自己的平衡没有天生的免疫力，它从来都不是自动发生的，必须通过不屈不挠的努力来维持，麻木和忽视不算，通过有意识地使用意志来获得宁静是天秤座工作的本质。

天秤，平静的符号，将自己的神经绷紧得像小提琴琴弦，天秤的艺术并不是去放松它们，而是看到它们从未被拨动，变得脆弱敏感，但还是能平静。天秤必须学会去意识情绪失衡的初期症状，永远不要让压力形成连锁反应，在它开始之前停止它。

```
约会已经晚点了十分钟，电话在不停地响，透过窗子还看到一个当初让你很烦的大学室友走到你的门厅前，想来一个突然探访。进行一次深呼吸，静心五秒钟。
在山间小道上转一个弯，从灰暗的松树林迈进一片广阔的草地，抬眼看去，数英里的山脊上都是蓝色和绿色，还有黄昏下很嫩的粉红色、红色。
当我们看一幅美丽的油画，终于打扫完毕后看着自己的睡房时，或者看到一个芭蕾舞演员旋转通过舞台时，也会有相同的心理和身体反应。
```

天秤座通常会提升对艺术的敏感度，对美的欣赏和创造都会带来平静。当感觉这种惊喜和舒畅，不知不觉地，肌肉放松了，我们长舒一口气。外在世界中所感受到的和谐，会被解释为内在世界中的平静，天秤必须不断美化自己的环境。学习绘画可以达到目的，去那些美丽的地方比如山川或画廊也可以，整理床铺也行。

和谐不只通过五官进入意识，也通过感受而来，感受往往有赖于跟别人关系的质量。对其它星座来说，友谊和婚姻的问题都没有这么重要，而天秤座只是一半，它所寻求的宁静来自它所找到的另一半。天秤的目标是建立深入和谐的关系，这是一个高强度的成长领域，要在这方面做功课，还可能在这里发生很多激烈的起伏。要学习去妥协自己的习性，但不要妥协自己的本质。危险在于，天秤在一段关系中如此在意保持和谐的外表，以至于所有真正的宁静被丢失了。

###### 优势

```
女性完整男性，光明完整黑暗，邪恶创造美好，美好为邪恶带来意义。
告诉天秤政治左翼人士拥有所有的答案，天秤会颂扬右翼人士的智慧。告诉他保守派会拯救这个国家，天秤会指出自由主义的美德。
玛丽认为杰克是自私的，杰克认为玛丽是操纵的，他们的朋友意见不统一，他们俩中的一个必然是对的，但是是谁呢？只有天秤理解他们都是对的，真理在两者的均衡中找到。
```

不只是艺术敏感度，不只是亲切感或社交场合的优雅，而是一种更深的东西，天秤的宝藏渗透在所有美德中。天秤座的和谐感指的不只是颜色、形状或个性，而是所有整体都由互补的两半组成的意识。直觉地，天秤感到金子必须和铅均衡，真理永远被两个相等而相反的真理所均衡，每个真理都包含了两个 “半个真理”。天秤无法满足于只有半个真理，比其它任何星座都更能够忍受矛盾，他不需要世界有意义，天秤能接受全部相对立的哲学、人、选择。给天秤座塞任何僵硬教条的意见，他立即就会提出相反的观点。

###### 劣势

```
想要上医学院就意味着无法写关于克里特岛的诗，想要结婚意味着无法继续单身了。
```

真理本就是模糊的，如果能接受这一点，感知就更清楚了。广阔的心智可以同时保持两种可能性，对矛盾的容忍是智慧，但生活却狭窄得多，每个决定都是明确的一步，退路上的桥必须被烧掉。这是非常糟糕的负担，天秤站在十字路口犹豫不决，震惊地意识到没有一条路是完全绝对正确的，唯一能做的就是扔一枚硬币，然后无比热情地投入这个选择，好像没有其它选项一样。

我们无法永远保留选择权，但天秤却有可能尝试这样努力。天秤四处游荡，甜美地微笑，他们不会有任何敌人，因为他们不会采取任何立场。但在平静表面之下，压力正在积累，有什么从根本上来说不对。没有威胁，没有压力，没有问题，任何事情都没有在发生。如果没有承诺，生活不过是漫长的等待，一只古老的钟在某个地方滴答滴答走着，不管是否做决定，生命都在继续。

### 水星-思维

当行星离太阳越近时，轨道就被拉得越短，它们也运行得越快。即使是跑得最慢的冥王星，速度也有每秒3英里。在交通法庭上这种速度都被称作是恶性超速案件了，但对行星来说，这个速度简直就是原地不动。

没有行星比水星更靠近太阳，也没有行星像它那样快。水星以每秒30英里的速度在太空中呼啸而行，每88天就绕太阳一圈，冥王星则需要248年才能绕太阳一圈。那些水星型的人感觉就像是：在一个大家都每分钟转33圈的地方，以每分钟78圈的速度在运转。

当水星在舞台中央时，我们的脑子清晰而迅捷，我们的智力以最大限度在运作，我们也心惊肉跳得像一个在看肺癌节目的吸烟者。那些思维可能很遥远和抽象，也有可能只是购物单的简单背诵。它们像雪崩一样在头脑中不停翻滚，速度每秒30英里，还常常绕圈。

速度既是水星的长处，也是它的短处。水星会比其它行星都更快速地吸收各种印象，它对这个世界中的形状、声音还有概念都充满如此深的迷恋，它可以像镜子一样清晰而不评判地处理这些印象。

但是这些感知也可能完全没有被消化地通过头脑，就像是一个不识字的中世纪男爵的图书收藏，它们只是在头脑的书架上不停积累。失控的水星可以运转到警戒线的速度，让一个人变得对很多色彩斑斓而毫无意义的琐事着迷上瘾。

```
你是否认识这样的人，他能够告诉你贝多芬在哪一年写了哪首交响曲，但是在听那些交响曲时却从不会享受地微笑和用脚打拍子？

```

就像它比其它任何行星都要快一样，水星对应我们意识中所有快速的记录：思维，我们头脑电路每一秒钟的运作。这就是那个掌管我们脑中图书馆的行星，它的作用就是思考、了解、推理。

在古老的神话中，水星是天神的信使，它跟口头还有书面的交流相关，一个受到强大的水星影响的人往往是能说善道的思考者。不过语言是之后被文化人为创造出来的，是一种为我们被混淆的感知带来秩序的方法。

水星是更加基本的心智的行星，对应线性的、逻辑的功能。纯粹的水星能量是头脑中无组织的、自由联系的想象游戏，是意识对感官刺激的反应。水星的功能是观察和信息传递，它关心的是更在眼前的问题，不需要知道生命的意义。

如果没有谦卑和谨慎，我们可能只看到自己想要看到的东西，有选择性地收集自己的感知，总是努力去支持那个自我在其中感觉最舒服的模型。这颗行星能够用事实证实任何事，这种合理化和防卫就是水星的另一个陷阱。

如果它无法做到这一点，它也能够以非常快而愤怒的方式来说话，以致于我们会漏掉它论据中的漏洞。我们的思考和认知功能很容易被恐惧和个人倾向所包裹，而成为我们的自我（ egos）。

从地球上看，水星总是跟太阳连在一起，在空中，它们两个永远都离彼此不远。感知对头脑来说，就像食物对身体一样，头脑围绕着自我转就像是水星绕着太阳转。水星的任务是将自己从太阳的应声虫变成他的大臣，水星必须只是收集原始的信息，并将它传递给太阳。

>**来源：**  
>[内在的天空](读书/神秘/内在的天空.md?id=内在的天空)

在神话中，墨丘利(Mercury)是众神之间的使者，负责搜集事实和传播信息，所以星图中的水星也与心智、各种信息交换的活动有关，例如写作、演讲、教学和旅行。墨丘利出生之后被安置在摇篮中，但没一会儿就离开了摇篮，想找些新鲜事来做。因此水星在星图中的位置，就代表我们最无法安静下来、最想探索，或是最好奇的生命领域。水星就像是一个成长中的小孩，对世界上所有不同的事物着迷，很渴望去触摸、定义、发掘事物，更想要了解生命是如何运作的。在神话中，墨丘利总是一副年轻的模样，青春不老。所以我们可以从水星的宫位看出自己会在哪个生命领域中永葆青春，并且抱持着一种开放和新奇的态度。

墨丘利是仲裁者和交易者，面对不同的天神时各有一套说辞。他出生后的第一件事就是向奥林匹斯山的十二位伟大天神提出交易。他还是一个恶名昭彰的小偷，他常从所有天神那里顺手牵羊（例如阿波罗的牲畜和阿佛洛狄忒的腰带），同时也十分擅长模仿，可以从每位天神身上学到一些特质。所以我们也可以按照水星落入的宫位，看出自己在哪个领域中最具弹性和变化，同时又会有些善变或浮躁不安。

每当你觉得总算逮到墨丘利时，他就会马上溜走，然后又带着另一套说辞出现。心智本身就像天神墨丘利，很会玩把戏。心智赋予了我们客观和分析的能力，但也时常会扭曲事实，戏弄我们，让事实看似合理。中世纪的炼金士曾经描述过一个名为“墨丘利斯”(Mercurius)的人物，他被认定是“创造世界的灵魂”和“隐藏或囚禁在物质中的灵魂”。具有讽刺意味的是，他负责创造世界，但自己却被困在其中。在某种程度上，是我们的心智决定了这个世界的状态，20世纪德国科学家沃纳·海森堡(Werner Heisenberg)毕生研究量子力学，发表了“测不准定律”，其中就提到：“观察这个行为本身，会影响被观察的事物。”

丘利将神的信息传递给凡人，他在某种意义上代表着所有人都会经历的一种过程，那就是在高层智慧和日常生活、意识和无意识，或自我和环境之间，搭起一座沟通的桥梁。心智虽然可以让我们与其他人区别，但也可以用来拓展意识，进一步地觉察万物之间的关联性。心智可以用来切割，也可以让我们与万物融为一体。我们可以从水星落入的宫位看出，自己会在哪一个生命领域中画地自限，或是会向外伸出触角，更加了解自己和他人。近两千年前的哲学家爱比克泰德(Epictetus)曾说过一句值得深思的话：“我们不是被事情困扰，而是被自己对事情的意见所困扰。”在水星落入的宫位所代表的生命领域中，我们不见得能改变世界，但却可以改变自己对世界的看法。

水星与双子座和处女座有关，这也代表水星的两种互补特质。水星的双子座面向就是擅长将零星的信息拼凑起来，联结生命的不同领域，处女座面向则是将事情抽丝剥茧，仔细地分析其中的每一个环节。双子座会引出善变、善于沟通和浮躁的特质，在与这个宫位有关的生命领域中，我们会倾向于“随时随地”出发，凡事都是浅尝辄止，就像是四处采蜜的花蝴螺一样，很难安定下来。处女座则会凸显出水星精准、爱挑剔和注重细节的特质。双子座对于许多事情都只是一知半解，处女座却会对少数的事情有深入的认识。在双子座落入的宫位中，我们会为了知识而追求知识，但是在处女座落入的宫位中，却会为了能实际且有建设性地应用知识，才去汲取知识。

```
作家奥斯卡·王尔德的水星落入第三宫，双子座在第十宫的宫头，这意味着沟通的行星和宫位（第三宫）与工作及事业（第十宫）产生关联，但是他第三宫的水星也跟上升点有关，因为处女座位于第一宫的宫头，所以他会用犀利的理解能力、略具威胁而又一针见血的机智（天蝎座水星在第三宫）去面对人生（第一宫）。
```

#### 双子座↑+↓射手座

上升点落在双子座，这个人应该是带着好奇心和求知欲去面对人生，渴望能了解每个人、事、物是如何运作的。多才多艺和顺应通融是双子座的两大资产，但他们最好不要兴趣过于广泛，否则可能落得“学百艺无一精”的下场。上升点在双子座的人痛恨失去其他的可能性，而专注在单独一件事情上面，就意味着排除了其他的可能性。

他们可以通过发展写、说或是其他形式的交换意见能力，来确立自我意识，并增加对环境的影响力。在某种程度上，他们必须散播资讯，也就是撷取某个地方的特定想法或态度，然后将这些东西沉淀之后，再应用在其他的地方。如果星图中其他部分具有水象或土象的特质，他们就必须发展出分析、抽离和客观的能力。

但如果星图中其他的部分已经具备风象和火象的特质，上升点在双子座的人就会更加浮躁不定，很难长时间停留在同一个地方，也没有足够的时间去真正深入地了解一个领域，而很难成为一个领域的权威。这里的危险之处就在于，他们常会太快“进入大脑”，而与身体和情感失去联结。

下降点是射手座，可以透过伴侣关系去了解另外一个人的哲学架构，借此去认识、探索并阐释生命。对他们而言，最好的伴侣应该是能扭转双子座的倾向，让他们不会迷失在想法的迷宫中，或是一堆无关紧要的细节之中的人。射手座的广阔视野和热情抱负，可以帮助上升点在双子座的人锁定目标方向，而非不停地绕道而行，另一半可以弥补上升点双子座所欠缺的整体目标感。

上升点在双子座的人多半具有柔软、敏捷的身型和修长的双手，讲话时常会加入手势的表达，同时也很擅长将拆解后的东西重新组合起来。这种人时常对许多情形有“双面看法”，也会随时随地注意形形色色的人，通常都很擅长模仿他人。

```
爱尔兰剧作家萧伯纳善用敏锐的心智和文学才华，将自己的哲学和社会理念发扬光大。他就是上升点在双子座、下降点在射手座的极佳典范。《隐藏的说服者》(The Hidden Persuaders)和《攀权附贵者》(The Status Seekers)的作者万斯·帕卡德(Vance Packard)也善用了上升点双子座搜集事实资讯的能力，他在探讨当代社会的著作中，觉察并揭露了更广层面的潮流和趋势（射手座）。
```

#### 双子座

```
砍下自己的头！将你的整个身体溶解为图像，看见、看见、看见！ 
——鲁米
```

###### 形象

双胞胎，兄弟，姐妹，灵魂伴侣，爱人。两个存在体无可救药地绑在一起，对彼此都充满好奇，但也被一面无法穿透的误解之墙所分开。一千个拼图被炸成了一百万个碎片，散落在地如纸屑一般，两个疯狂的天才准备攻克这个难题。他们说话，倾听，扫描对方，寻找线索，有意义的手势和隐藏的暗讽，没有东西会逃过他们的眼睛，没有一个细节是过于细微的，没有一个足迹是被忽略的。


###### 主题

双子生来就要感知，将自己沉浸在观察之中。我们通过倾听、感觉和见闻来感知周围千变万化的风景，与世界交流沟通。就像一个孩子只有十分钟要去扫荡一个玩具店，有这么多的东西去看，去了解。双子匆匆地走过各种体验，一分钟也不能浪费，有时候是手忙脚乱地，通常没有一个明确的主要计划，但他总是带着浓厚的兴趣。

机敏的智力是很基本的，但这个星座的主旨是纯粹的感知，不是思考，而是看见。双子是记者，而不是哲学家。未经消化的，感知到的事实是他的食物，而不是这些事实的意义。他只是想看到、想见证这个世界。揭开这个世界的奥秘，收集所有的线索，看到一切事物，就是双子的目的地。在此过程中，一些想法和理解会升起，但它们不是重点。


###### 课程

看到一切是双子的任务，就像用一根针来勘探大海。给一个人百年生命，计算机的头脑、X光的视力和电子耳、还有四岁小孩一样高度活跃的能量，他能看到什么？他能看透多少生命？这是不可能的。双子的关键策略是活在最高强度和最多变化的生活中，将可预测性和无聊当作不可饶恕的罪过，双子必须保持在运动之中，永远不让头脑停止成长。

```
医生不喜欢看到一个靠信念而成功疗愈的人；种族主义白人会觉得一个聪明的黑人物理学家对自己一点用都没有。
```
我们都会在头脑中创造一些关于宇宙是什么样子的图像，寻找证据来支持它，而不喜欢与之不符的体验。双子必须违反这个本能，允许宇宙不合理，以完全的接受状态保持很广的敞开度，不管观察多么不可理解，努力去看清楚更多，而不是思考。如果他们是迷惑的，那只是表明所收集的信息超过了理解能力，说明走对了路。

生活是收集体验的一种方式，世界充满了体验，充满了人，每个人都有一套独特的感知系统，将它们从较粗燥的观察消化成更加精细的思想。想要得到周围人的这些珍宝，双子知道只需去询问。双子生来就要说话与倾听，机敏的心智和口才，精通说话的艺术对双子而言要比用心倾听来得容易。我们要说出自己知道的东西，但是有时会听到一些自己不知道的东西，双子必须记住沟通是双向的，只有当讲话能够加速倾听、吸收世界的过程时，它才是有用的。

出生盘较硬和直接的双子，可能会打断别人，替他们完成剩下的话。出生盘更柔软和敏感的双子身上，这个模式可能没那么明显。他们或许很安静，对别人的看法显得很感兴趣，但他们的想法早已经跑到前面去了。虽然眼神接触并频频点头，但是并不知道在说什么，心里早已经决定了对方要说什么。


###### 优势

好奇，对世界的惊叹之感，对每一滴露珠、每一滴泪水、每一朵雪花的孩子般欣赏。好奇给双子带来不停止的动力，一旦热情被点燃，双子就像充满活力的流星一样划过生活。很少有星座能够如此毫不疲惫地生活，睡这么少的觉，有这么多要看的，这么多要知道的，一辈子要过三辈子的生活，没时间偷懒。

身体的活力运作在头脑层面带来了智力，双子将所有的好奇、精力结合在一起，建造了巨大的未经消化的原始信息的图书馆，在其中以光速扫描，寻找连接，找到共同点，注意到差异。没有哪个星座像双子那样对信息如此饥渴，并那样精于收集信息、找到关联、并传播信息。对双子而言，宇宙的基本成分不是分子和原子，而是信息。他们能探测到别人的想法，过滤掉虚假和编造的东西，找寻到金子般的洞见和水晶般的体验。


###### 劣势

最纯净的双子只是观察，而不会评判，就像一面没有意见的镜子一样。双子永远在犹豫，不会给出一个不够成熟的结论，总是怀疑有更大的构架与终极的真理，双子编织了一张线索的网，他们知道任何事情都可以被事实证明。他们没有自大和教条的空间，这些东西将他们与混乱和神秘隔离开。但双子适应性强的特征跟更低等的动机结合在一起时，他们的变通性就会制造出恐怖。

当被迫要面对一个令人不愉快的真相时，他们的头脑可能会重新组织信息。他们并不会撒谎，而是通过一些选择性的强调、策略性的沉默、以及一排集中的话语来阻击别人。不管是如何的不真实，他们蒙太奇般的说法会显得很有道理，至少足够说服任何人忘了到底是怎么回事。不管是正确的还是错误的，他们很少会在辩论中败下阵来，一旦双子开始为自己辩护，只有最狡猾和一根筋的人能够看穿他们所编织的迷宫。

双子一刻不停的嗡嗡的能量不可避免地会流入神经系统，这种无法放松常常会引起失眠，造成他们特别容易激动，并加剧神经质的唠叨。如果头脑中累积的压力没有被释放，外在情境会开始反映同样的模式：繁忙、压力过大、最后就是情绪衰竭。跟一个有深刻洞见的伙伴进行安静的谈话能够帮助放松，没有智力的斗争，没有言语上的交战，只有简单的心对心的交流，没有人记笔记，也没有人想向对方推销什么看法。

运动也能够帮助双子释放压力，走路、跑步或友好排球赛，但是俯卧撑不行。双子来到这里不是为了变得无聊，而是是为了收集体验，通过无法满足的好奇心，将各种体验塞满自己的一生，让生命的奇迹通过他们的心而运作。但是他们所骑的是一匹野马，能将他们带到远方的地平线处，也可能会在原地不停地转圈、乱动，哪都没去成，纸上谈兵成了很多双子的墓志铭，这匹马需要一位驾驭它的骑士。

#### 处女座↑+↓双鱼座

上升点落入处女座的人会透过智力的分析、辨认、自我批评，或是更加明确地确立自我，来培养自我的身份认同。这代表一个人必须在选定的工作领域，或是发挥创意的领城中，培养更加完美的技巧和熟练性。作为土象星座的人，他还必须有目的地应用知识，让知识更具有可以运用的价值、更具有生产力，或是更能带来帮助。上升点落入处女座的人必须学会关心身体，注意身体的机能是否能和谐地运作。另外一个功课是，必须务实地管理日常生活中的世俗事务。

也与所谓的“正确地吸收经验”(the proper assimilation of experience）有关，应该把这个过程应用在生命经验的消化上。经验就像食物一样，必须被一再地咀嚼，然后被消化和吸收。身体内的新陈代谢就是在区别什么是有吸收价值的，什么是有毒害的。他们可以透过全面性地分析自我和生活，再从经验中萃取出有利于个人福祉的部分。他们必须承认、并且辨识那些没有助益的部分，再把那些部分清除和消灭。长时间的愤怒或消极的态度，常会导致精神呆滞、衰弱，并且中毒，迟早会波及身体。

### 木星-乐观

#### 射手座↑+↓双子座

射手座的符号是弓箭或一个半身是人、半身是马的怪兽，上半身是人的躯体，手拿弓箭瞄准天空，下半身则是马，马蹄坚定地（或是雀跃阔步地）踩在地上。上升点在射手座的人时常得在怪兽和天神之间，维持一种不稳定的平衡，这也代表他们在面对人生时的两难。他们天性中有一部分渴望着更高的境界、高贵的理想和崇高的成就，另一部分却受到直觉的生物本能所驱策。生物本能可以和更高的梦想同步前进吗？上升点射手座经常为了现实与理想的差距而感到痛苦，所以必须找到一些创造性的方法，解决这种分裂的需求。

另外一种与射手座有关的形象就是追寻者，他们认为凡事都有前进的空间，总是还有其他的事物值得去追求。上升点在射手座的人最好把生命视为一趟旅程或是朝圣之旅。有时旅程的本身会比最后到达的终点更为有趣。这也就是古老的座右铭：我看见目标，我达到目标，然后去追寻另外一个目标。在外表特征上，上升点在射手座的人常给人一种浮躁不安的感觉。有些人的特征是嘴阔或是“如马般”的露齿大笑，仿佛一定要有足够的空间让他们畅所欲言。基于这个星座的扩张本质，当一个人的上升点落入射手座时，也可能会有体重超重的困扰。

上升点射手座相当于一个火象星座落入角宫，代表他们可以产生一种狂热，同时需要找到一种渠道或是一个重心，传达自己的能量和热情。上升点在射手座的人天生就具有激励他人的本领，前提是他们没有将人群抛到脑后，过度地展现自己。他们也有能力赋予世俗的事件一种象征性的意义和重要性，让这些事件不再被视作是单一的存在，而是与更大的真相或宇宙法则有所关联。

上升点射手座的危险之处在于过于自负、表现过度，或是浮夸不实。就像是希腊神话中的悲剧人物伊卡洛斯(Icarus)，他们可能飞得太高，然后会直接坠落到地面。有些人可能会习惯地过着入不敷出的日子，有些人则是一直活在充满可能性的世界里，却不曾将梦想具体地落实。我们还要参考星图中其他部分的土元素和土星的影响，才能判断问题的严重性。可以从射手座主宰行星木星落入的宫位，看出他们相信自己能在哪个生命领域里实现梦想。他们可以在这个领域中扩展他人的视野，但也可能是自己扩张过度和毫不节制。

双子座落在下降点是一种必要的对照。他们可以透过伴侣关系，近距离地审视自己的基本生存法则和模式，这些都是他们透过直观所获得的。双子型的伴侣非常适合他们，因为双子座可以找到精准的字眼，表达射手座广泛的概念和感受。双子座可以分析、质疑射手座的结论，迫使射手座更全面地去思考一件事情。双子座可以告诉射手座，如果想要实现目标和抱负，当下应该采取哪些具体的行动。射手座也许会决定周末一定要去度假，但很可能是双子座拿起电话簿，找到旅行社的电话。就像其他上升点和下降点的组合一样，我们可以从另一个人的身上获得下降点的特质，或者为了追寻自我的完整性，而从自己的内在发展出这些特质。

###### 形象

手指因张力而微微发抖，弓箭手放开弓弦，曲线弹回原处，将带着羽毛的箭杆射入空中，弓箭以看不清的速度在空中穿行，朝云端飞去。如果它有眼睛，会看到弓箭手的身影越来越小，慢慢变成景物中的一个小点，底下的景色像地图一样摊开，简单而清楚。如果这支箭被赋予了头脑，就像在山峰上的人被活力的能量所驱动，会有一些关于全局和体验的想法。谁将我投射到这明亮的天空中？为了什么目的？我在这里的位置是什么？我要去向哪里？我存在的原因是什么？

###### 主题

就像那支飞行的箭，心智将自己投射到新的地平线上，通过收集不熟悉的体验来扩展意识，这种扩展就是给射手提供生命的血液。吉普赛人的扩展来自物理行动，从一个地方移动到另一个地方，进入新的异域环境中。学生通过智力吸收新的事实和观点，哲学家则用直觉努力意识宇宙的基本原理。

射手的目的地是意识到生命的终极意义，在宇宙设计中找到自己的宿命，发现真理，要达成这些非常高远的目标比爬上一根旗杆要难，也是最难从实际层面来定义的。射手没有一个清晰的终点，只有无尽的过程，是一个移动的、流动的存在状态。生命的真相是永无止境的搜寻，意识到人的基本任务是找寻意义，在这无常的世界中，如果安全感和安定感阻挡了无尽探索，牺牲它们也不算什么。

###### 课程

第一步是成为吉普赛人，释放将人捆绑在某个特定文化，以及与之相关的价值和习惯之中的束缚，对陌生的思考方式保持敞开。面对一个基本运作方式完全不同的社会，放掉自己从小到大所形成的评判和优越感，旅行只是用来实现敞开的方法，教会我们以不同的角度来看生活，就像按下进化火箭的点火器一样。

并非单纯地收集很多异域的护照章，而是通过意愿和敞开来接触不一样的文化，并由此扩展意识。让富人跟穷人接近，白人跟黑人接近，即使只是一次城市里的旅程，跟一个在这个国家生活的人交朋友也一样有效。

心智首先对新体验敞开，才有了新的学习机会。正规的教育可以扩展意识，射手喜欢课程和讲座，但书本也有一样的作用，即使不是机构化教育所支持的书本，任何迫使我们以新鲜方式来看待生活的体验也同样有效。射手以一种冒险的态度生活，放弃所有安全的概念，只要看见自己想要躲在它后面，就必须愿意放开任何一个想法或者主意。他必须毫不犹豫地向前，期待奇迹，随时准备凭借信念纵身一跃。

哲学家的路有赖于吉普赛人与学生收集的洞见，没有它们，他不过是一个干枯的老学究，装了一脑子没有生命的假设。哲学家直觉跳跃的结论，必须有经验的支持，否则它没有现实的根。将注意力从眼睛和耳朵上，转移到人最深处的感知功能上，直觉地努力理解生命的整体性，并找到我们在其中的位置。哲学家吸收基督教、佛教、存在主义等等的所有一切，不断收集更多体验，不停修改和深化对生命的看法，但永远不会让任何一个系统替他思考。

###### 优势

射手极度热爱自由，没有哪个星座如此害怕束缚。没有个人自由吉普赛人就枯萎了，学生也会被无聊的常规所困住，哲学家无法获得更大的视角，或在不同境遇的现实中进行验证。

不管生活将什么扔给射手，他都准备好了热情、乐观、冒险精神，他有适应性和弹性，能从任何逆境中重新反弹。射手有一副抵抗哀叹的盔甲，虽然还是有可能悲伤和抑郁，但这些都只是暂时的，任何时候只要有冒险和新的可能性出现，这些状态就会烟消云散。

射手不是天生就知道生命有什么意义，但是他确定生命一定有意义，对理想的信念是毋庸置疑的假设。就好像他来到世上就在脑中带了一个空的圣坛，他会本能的在那里供一个偶像。理想可能会以一千种形式出现，耶稣或文学，也可能是世界革命，不管哪种形式，射手都会以它来指导生活。以他的自由方式，他会选择自己的神，不管因此有怎样的外在压力，他还是会选择。

没有哪个星座有如此坚定的原则性，不惜任何代价，不管是否现实，射手都要去做正确的事。这种正确是自我定义的，并不是每个人都能赞同它。但一旦知道了射手的神是什么，他的行为就像行星绕太阳的轨迹一样可以被预测。

###### 劣势

生活有时候很黑暗复杂，充满了戴着天真面具的死亡陷阱，谨慎可能会救我们。射手明亮的眼睛里闪烁着信任的热情，犹豫和怀疑是很陌生的，过于乐观，过度扩张，还有错误的判断都常常令那支箭半道落入陷阱。就像一只在洲际高速公路上奔跑的金毛犬，他认为这些过来的车都是自己的玩伴，结果常常被碾碎在生活的车轮之下。

强大的信念、幽默感、敢于冒险的品质帮助射手变得更加完整，但如果它们没有被生命的脆弱和短暂的感觉、生命多么容易消逝的意识所均衡，拥有光明精神的射手可能会成为悲剧。射手常常会过于信任那些不稳的步骤，可能会在一个看不见的弯道上继续向前，以信念的名义，跟随一个自命的上师到集体坟墓里去。因为他大大咧咧的态度，射手甚至会为了永恒的浪漫，而将至关重要的自由抛入水中。急于结婚，后悔莫及的亲密关系，是很多不快乐的射手的墓志铭。

### 土星-天命

木星会为一个宫位带来扩张和乐观的气氛，土星则会引发几乎完全对立的感受。在土星落入的宫位所代表的生命领域中，我们不相信生命的本质是仁慈而值得信赖的，反而会期待困难、失望和限制的发生，因此会戒慎恐惧地进入这个生命领域。我们在木星的宫位里，常会沉浸在自由和无限的可能性之中，到了土星的宫位却会面临许多限制，还会感受到一种无法摆脱的责任和义务，以及一些生命中“应当”去做的事情。

土星的面貌之一就是古老的暴君，天神克罗诺斯(Cronus)担心会被自己的小孩篡位，竟然就把小孩吃掉了。我们在土星所在的宫位中会因为保守的心态或是恐惧，而无法放任创作的冲动恣意奔驰。我们惧怕未知的或任何未尝试过的事物，所以即使现存的一切已经不再美好，也仍会在这个生命领域中维持现状。在土星的领域里，我们会自我批判、自我抹煞，总是担心自己会出错，所以为了安全起见，往往会严格地限制自己的行动。我们就会像克罗诺斯一样，不停地限制、评判和审察自己，然后将象征个人创意表现的小孩，狼吞虎咽地吞下肚。

克罗诺斯的形象总是挥舞着一把镶刀，他的一生会让人联想到一句古老的谚语：“种瓜得瓜，种豆得豆。”他先阉割了父亲乌拉诺斯，然后篡位，随后又在儿子宙斯一手策划的叛变中被推翻下台。土星代表的是不偏不倚的正义法则，我们如果忽略或逃避土星的要求，就会感到不安而痛苦，但如果愿意付出努力，坚持不懈，展现无比的毅力，土星就会给予我们应得的回报。

土星也与智慧老人的原型有关，他就像是天国学校的老师，会用痛苦来提醒我们有哪些方面需要注意和发展，如果逃避这些功课，痛苦只会有增无减。如果没有落实土星的宫位代表的生命功课，往往会感到痛苦，但却会为了掩饰或减轻这种痛苦，一口否认这个生命领域的重要性。但是这种生命的残缺不全，迟早会给我们迎头痛击，而且会不偏不倚地正中要害。

我们应该用心聆听土星想要教导或意图展现的功课，同时逐渐地将这种不舒适感，转化为与日俱增的完整感、坚定感和价值感。只要我们能面对土星的挑战，就可以让自己变得更强壮，从中获得更多的知识和成就，最后反而可以在那些曾经难以掌握的生命领域中变成导师。德国诗人歌德（上升点土星在天蝎座）曾用一句话精辟地点出了土星的意涵：“大师都是在自我的限制中展露锋芒的。”土星就如芒刺在背，会不停地刺激我们去发展一些才能和特质，而这些通常都是我们不愿意花精力去培养的东西。

我们会在土星落入的宫位中展现山羊的特质，山羊努力地爬上山头，中间会经过许多上下坡，但最后一定能够抵达终点。山羊会先确定这一步已经站稳了，然后才踏出下一步，所以我们可以在土星的领域发现以下的正向特质：慎思、老练、责任、义务、坚韧不拔及正面地接受现实。

木星给我们带来的是信念的感觉，不过它常常用一千个拐杖来支持我们。“我当然有信念，生活很美好。看看我的银行存款，我的性感伴侣，我的崭新奔驰⋯⋯ ”土星对待信念的方式则不是这样的，他将我们的所有拐杖都拿走。土星向我们展示黑暗、不可能、失败，然后问你：“你还有信念吗？”

它是对我们拥有天命的信念，是对生命、对自己、对我们的梦想、愿景、理想的信念。只有当这些愿景在绝对不可能的面前，都能够无需任何帮助而独自站立时，我们才能够说自己是拥有信念的。在此之前，我们都还在进行练习而已。

隐士常常被用来作为土星的象征，不管土星待在星盘的哪个地方，那里都是我们必须独自行动的一个领域。在这里，我们无法依靠任何人，除了我们自己，甚至连回报也要由我们自己来提供。只有土星才能带给我们从自我认识和自我尊重中所升起的自我满足，没有它们的话，喜悦就像是风中的纸张一样没有根。

如果没有学会土星在星盘中所编织的功课，我们就会四处漂移，迷失自己的方向。如果我们令它不快，生活会变得空虚、毫无意义，我们会落入愤世嫉俗和绝望中，感觉抑郁、忧郁、失败、孤独以及沮丧。但那不是土星的目的，没有一颗行星是来伤害我们的。

这颗戴着光环的行星常常被看作凶星，像另一颗所谓的“凶星”火星一样，土星想要将意志聚焦。它教我们最难的艺术：如何完全依照我们的喜好而行。而土星教我们的最重要的美德就是自律，如何让我们的意愿驾驭我们的恐惧、懒惰、情绪。

```
一位年轻的梦想家，她有英语硕士学位，可是却没有用它来找工作。她的志向是成为一位作家，不过至今为止这一直都只是一个幻想，她什么都没有写过。在过去五年里她都在等待，等待着什么的发生。突然有一天她发生了改变，她发现自己正在变老，于是变得有些害怕。她意识到要么就是现在，要么永远什么都无法发生。
她写了一本小说，这本书反映了她的特质，很梦幻、很浪漫，一本哥特式小说。突然之间，喜从天降。她卖掉了手稿，小说大卖，全美国每个零售店的柜台里都有这本书。三十年过去了，三十年里，她出了十五本书，每本的名字都不一样。她变得有钱而出名，你可以在《人物》杂志上看到她的脸。
岁月改变了她，她不再是一个精神恍惚的天真少女，而是一个有远见和信心的成熟女人了。但是她的书并没有反映出这一点，而她自己是知道的。她有了一部新小说的灵感，一部非常不同的小说。更深刻、更严肃，一部会让她自己自豪的作品。
她的出版商对此不抱信心，但她却一直坚持写作。写到一半的时候，她去看医生。癌症，还有三个月可活。而这本书还需要六个月才能写完，不可能了。她哭了，她写了遗嘱，计划着进行环球旅行。一个星期过后，她跟自己进行了一场很长的谈话，然后她耸耸肩膀，又在自己的打字机前坐了下来。
她一直是作为一个作家而活着的，她也将作为一个作家而死。那就是她的命运，从那里逃走就像是从更深的自我中逃走一样。她的著作永远不会完成了，但是谁管它呢？那没有关系，关键是过程。写作这个行为本身就是真正重要的，没有成功或者失败。在那个瞬间，这个女人通过了所有行星中最强烈的考验，她成了土星的主宰者。
死亡将阻止这位作家为自己的书获得任何的称赞和金钱，没有一个人能够看到它。对世界来说，就好像她从未写一个字一样。但是她还是去写这本书，只是为了完美无缺地成为自己，从而获得满足。从现在起她将只为一个人而创造：她自己。她可能会爱，但是她永远不再需要任何其他人的赞同了，她的自我是完美的。
```

#### 摩羯座↑+↓巨蟹座

在神话学中，摩羯座的主宰行星土星具有分裂的人格。一方面，他把自己的父亲去势，还吞食了自己的小孩。从这个层面来看，土星代表的是压抑法则，也就是挑剔、冷酷和严厉。但在罗马神话中天神撒登(Saturn，也就是土星)统治了整个“黄金时代”，当时的农神节(Saturnalia)是一个放纵、感官享受、丰裕和沉溺的庆典，因此也代表着丰饶的季节。上升点落入摩羯座也反映出土星本质的二元性，意味着我们必须同时考量这两种生命面向。

上升点在摩羯座的人时常会觉得有一位严厉的父亲在监视自己，期待自己会顺从听话，并且能有具体的成就。他们必须在一个明确、限制的范围内，务实而有成果地利用前一个星座一射手座的精力和热情，却不任由自己随波逐流，或是被突发奇想的热情冲昏了头。他们为了达到“内在父亲”的要求，必须谨慎地规划人生，按部就班地建构人生，然后缓慢地朝目标和抱负迈进，同时还必须斤斤计较能量的运用，而这一切都需要纪律和控制。

摩羯座主宰着膝盖，这意味着无论是内在或外在的形式，上升点在摩羯座的人迟早都得向更高的权威屈服，而这个权威也会对他们有所期待。就像希伯来族长约伯(Job)一样，上升点在摩羯座的人往往必须经历困难和挫败，才能够学会谦卑地接受法律、限制和结构。他们就跟基督耶酥一样，也许会在最后一刻怀疑起这一切的牺牲是否值得。上升点在摩羯座的人需要“让事物成为自己的财产”，并且获得某种程度的集体认同和尊重。他们可能会反抗，试图摆脱责任义务，但最后总是会发现唯有正视自己或他人的责任义务，才能让自己感觉更好。

作为土象的基本星座，摩羯座可以极有效率地运用物质世界的资源，可以透过发展一种熟练地处理现实事物的潜能，来获得满足和成就感。除了处女座，没有一个上升点的星座可以像摩羯座一样在混乱中建立秩序，或者找出让梦想成真的可能性。土象的摩羯座也代表“牧羊神”潘和“森林之神”萨梯(Pan和Satyr,两者皆是下半身是羊、上半身是人的希腊神话人物)，因此他们也具备了享受肉体感官和大自然的能力。也许是因为摩羯座能够充分觉察到现实人生的残酷，因此较之其他星座，更能够享受感官和美好的事物。

下降点是代表感觉的水象星座巨蟹座，巨蟹座象征的是柔软、丰满和圆润，它会对抗且很自然地调和摩羯座的严荀苛和缺乏弹性。上升点在摩羯座的人在外面可能显得很严肃，却经常会奔回家中，接受另一半的宠满或照顾。他们表面上看来很严肃或怯儒，但却能敏锐地感受到与自己亲近的人的感觉，并且愿意去改变自己以满足对方的需求。这就像一位严厉而又喜欢评判的父亲，最后会被全然接受又无微不至的母亲软化。如果我们在某一个生命领域里的某个方向发展过度，生命就会在其他地方补偿我们。

上升点在摩羯座的人就像一瓶好酒会越陈越香，随着时间的过去变得越来越好，越来越快乐。在外表上面，上升点在摩羯座的人通常都很精瘦又苗条，脸部五官的线条尤其明显。有时候他们就像是牧羊神的随从，看起来有些邪恶又淘气。

###### 形象

一只有鱼尾巴的山羊，一种并不存在的动物，海山羊在水中扑腾，在陆地上摇摇摆摆前进。在象征的世界里，海山羊失去所有的笨拙和脆弱，成为终极和绝对力量的象征。不管是在海中，还是在冷酷的高山上，他是两个世界的高手。海山羊横渡最广阔的大洋，攀登最险恶的高峰，在空气最稀薄的地方呼吸。没有任何东西能阻止他，他必须做的只是选择，一旦将强大的意志设定在一个目标上，他就是不可战胜的。

###### 主题

摩羯的目的地是内在和外在的合一，获得社会的认同，并且表达在社会中。在世界的广场上，摩羯必须按照自己的本质性格来行事，将个人天性跟他的大众身份结合。很多拥有权力的人活得像人质一般，被他们在大众面前的角色所限制，角色拥有世俗权力，但是他们没有。这不是摩羯的路途，世俗权力并不意味着荣耀，而是自由。

摩羯座是整合的象征，工作和他的生活不可分辨，两者合为一体。在摩羯的最高表现形式中，没有谎言，也没有虚伪，只有看得见的公共行为，和看不见的个人本质的完美结合。要达到这点，摩羯必须完全对掌声免疫，掌声会一直在他身边，诱惑他，挑逗他，无法从中逃开。虽然他的公共工作必须在大众面前进行，但他必须不为别人表演，一旦被社会认同的钩子钩住了，所有的一切都会失去。

要避免这种诱惑的力量，摩羯必须成为一个善于孤处的人。花时间跟自己待在一起是有效的进化策略，但重要是必须从内在寻找认可。想要成功，必须孑然独立，对成功和失败都无动于衷，宠辱不惊。摩羯的目的地是整合和独处，如果其中一点没有达到，另外一点也会随之崩溃。

###### 课程

在早期发展阶段，摩羯必须学会不需要任何人的认可，即使没有人欣赏或称赞他，也能对自己的思想和计划满意。摩羯并非没有爱或者冰冷的星座，爱和被爱不相违背，他必须做的只是避免需要另一个人。为自我满足找到一种实际的方式，独自散步、做白日梦、一个人滑雪或冲浪，任何一种能够独自进行的活动都有帮助，读书或冥想，任何一种能将独处从负担变为热情的嗜好。

当他能回忆起自己不需要他人的称赞和认可，他已经拥有了自我尊重，知道了自己是谁。只有获得这种内在独立状态时，摩羯才能安全地将眼睛转向世界。如果他提前这样做，荣耀和光辉会比太阳还厉害地让眼睛瞎掉。就像珠穆朗玛峰对夏尔巴人一样，摩羯是一个爬升者，这个闪耀的世界如此诱人而具有挑战性，他知道自己必须跑到顶峰，恐惧做出错误的选择。

当世界在他面前敞开色彩和戏剧性时，他必须退回到独处里，找到力量望向别处，将目光从总统服装上挪开并转向内，转向自己的根。但摩羯必须攀爬，必须面对世界，目光的转移并不是永远的状态，想要通过躲避世界来抗拒诱惑会一事无成。当他沉稳自信时，摩羯将目光冰冷的进入名人和名望的世界，头脑中只有一个目标，只去做那些自己天生要做的事情。

摩羯必须选择一个能够表达个人价值、兴趣和幻想的公共角色，也许是一份工作，也许是更加志愿性的事情，也许是一种公共服务，甚至也许只是一种个人爱好。不管采取什么形式，也许人们会为他喝彩，怀着敬意地看着他，叫他大师，也许人们会认为他一无是处。没有关系，两种情况下，他都必须像星云一样冷漠，唯一重要的是，他是否在做自己。

###### 优势

摩羯的必要工作是在社会中开辟出自己的位置，失败、不确定、长期只有很少的回馈，这些困难都是会折磨他的阻碍，但摩羯的耐心和自律可以让他经受不可想象的压力。摩羯的等待不是犹豫或摇摆，而是一种充满强度的静止，就像一只专注在八英尺外的老鼠身上的猫。不管压力有多大，一旦他下了决心，摩羯就会以无法比拟的自律一直坚持自己的路，没有什么事能动摇他。他也许能够感觉到恐惧、挫折、抵抗，但真正行为总是反映了他的意图，而不是情绪。

在摩羯的生活里，只有当幻想有成真的逻辑可能的时候，才有它的位置。如果有这个可能性，摩羯会非常有效率地带着绝对的坚持去工作，直到这个梦想成真。如果这种可能性不存在，他会像扔昨天的旧报纸一样扔掉它。摩羯有一种直觉的实用感，不管多么希望答案是 5，当他用2加2时，答案永远是4。

在坚硬的外壳之下，是摩羯钢铁般的意志，他的独立、坚决、耐心、准确无误的逻辑，都帮助他预测风暴，应对日常生活中的可能挫折，但同时又不会忘记梦想。他也许很慢，看起来就像在路上死机了一样，但最后，他会得到自己想要的。

###### 劣势

摩羯的道路需要非常大的自我控制，做出选择，然后行动，碰到失败，也会坚持。当正确运用在客观世界中，反映出他行动的本质时，能让他攀登任何高峰。但如果神经崩溃了，决定动摇了，误用的自我控制就会扭曲成灾难。

它不再与行为有关，主观地展现为摩羯对问题与发展的压抑情绪反应，变成一块没有情感的石头。当他的生活倾斜而失去控制时，摩羯可能会像黑曜石一样冰冷，往前走，摩羯会遭遇他天生独处态度的堕落形式，孤独。

孤独的、没有表情的摩羯还是令人敬畏，但当他失去了自己的方向时，扭曲的摩羯会力图决定周围的每个人的路。他变得独裁，像一个暴君，对人们采取一种施恩的态度，他本应该平等地分享自己的生命。

他在世界上寻找权力，就像一只饥饿的狼寻找腐肉，盲目地在外在寻找本应从内在寻找的赞同，不管去哪里，都想要扩展自己的权威。而他会成功，一步步爬上最高处，充满了坚持和决心。当他达到那里时还是不满足，所以他继续推动自己。他成了无情推动自己的工作狂，跟身体与疼痛的心失去了联系。死的时候，他是自己的角色和责任的受害者，他也许很有权力、很富有，却孤独得像一粒漂浮在星际虚空里的沙。

### 不可见行星 / 外行星

个性内的结构是可见行星的领地，它们完美无暇地勾勒出了这块地域的地图。七颗行星的系统看起来是人类心识的完美模型：个性、情绪、逻辑、爱、恐惧，什么都不缺。这个系统开始于太阳，生命活力之星，并且终结于土星，死亡和结束之星。生命周期完成了，似乎没有其它可能性了。

我们无法看到天王星、海王星、还有冥王星，只有当望远镜人为地放大感受时，我们才能够意识到它们。这些功能的正面显现永远无法被自动触发，在我们能够转化自己的存在之前，我们只能够看到它们的阴影。如果没有通过努力，那么这三个神秘的象征对精神来说就像它们对肉眼来说一样：不可见。

到底有什么处于个性的终极自我实现形式之外？死亡之外到底还可能有什么？这些新的行星让我们打破了自然、自我和常态的边界，我们来到了个性之外。这些行星都只是代表着可能性，只有通过对本性（self）进行有意识的工作才能够获得的一些超越性的、非自然特质。

>**来源：**  
>[内在的天空](读书/神秘/内在的天空.md?id=内在的天空)

三颗外行星无论落入哪个宫位，都会让我们在相关的生命领域里感受到更多的分裂、激进和转化的本质。我们不会用平凡、惯例或是简单的方式，去处理与这个宫位相关的生命事务，反而会用一种略为突兀的方法，与这个生命领域产生更复杂的关联。土星会借由其落入的宫位，让我们发现自己软弱、无法胜任或是不完整的部分，并凸显出我们需要努力和加强的领域，但外行星的影响力更胜于此。外行星会在落入的宫位代表的生命领域里，定期地挑战已经建立的结构，考验结构的稳定度和存在性。

外行星除了会让相关的生命领域变得更为复杂，同时也象征着某些形式的冲突、矛盾、紧绷和创伤，无论我们喜欢与否都必须做出重大的改变，或是转变自己的角色。崩溃是为了突破，希望最后的结果能让我们更宽广、更深入、更精微，更加延伸地去体验自我和整个生命。所有人的外行星都会落入同一个星座，因此天王星、海王星和冥王星对个人产生的影响，往往会表现在它们落入的宫位里。

### 天王星-变革

如果有一个占星符号可以对应物理上的“测不准原理”，那就是天王星，没有一个行星比它更加无法预测。即使在天空中，这颗富有个性的行星也按照自己的曲调来跳舞。大多数行星的赤道都差不多在黄道平面上，地球的北面也是木星的北面。但天王星的北面跟我们的北面是垂直的，这个行星并不在自己的轨道里进行陀螺式旋转，而是滚动。

天王星是个性和自由的行星，对应无法被驯服的特质，是我们内在那个拒绝被规矩压碎的部分。它很独立、叛逆、任性和多彩，对它来说，没有什么比进行反叛更加开心。语言、哲学、幽默、风格都不是个性（ individuality），这些特征只是从文化和经验中得出的面具。人格（ personality）的形成只是一个自然累积的过程，是由出生的环境所造就的。

个性则是更深的东西，它是未开化和自然的，是心识的天然结构的一部分。这一部分不会信那些我们所出生的民族里流传下来的神话，它不遵守这些规则，它不尊重任何权威。它对祖先、牧师、总统都没有尊重，它对任何的虚假都不尊敬。

阶级、权威、规则和制度是每一个文明的基础，没有了它们，我们会成为一群互相扔石头的洞穴人。但每个人都有一部分讨厌这些限制，它无法忍受被告知该做什么，这部分总是跟逃亡者、反叛者、打破规则的人相认同。当被压迫者挑战当权者时，不管理由是多么地令人无法理解，一句“我再也无法忍受了”总是能触动每个人内在同情的弦。

当我们发现了自己的个性并开始发展它，我们就会被整的很惨。自宇宙诞生以来，文化和个性、社会和天王星就一直是死敌。在它们的战争中没有英雄，也没有坏蛋，双方都纠正了对方的极端之处。如果文化赢了，我们都会变成只有两三个简单程序的机器人。如果天王星胜利了，我们会返祖成为野人。

>**来源：**  
>[内在的天空](读书/神秘/内在的天空.md?id=内在的天空)

天王星的名字来源于希腊神话里的天庭之神 Uranus，它在星盘上所处的位置会告诉我们，在哪个领域里我们的电板里装满了想要反抗的戏剧，在那里我们的本质会跟文化的限制进行战争。天王星挑战我们去获得自由，为了对自己真实，我们必须违反规则。但是违反规则时，我们陷入麻烦，我们被迫发展和保护自己的个性。

身为第一位天神的乌拉诺斯，除了要统治无垠无尽的宇宙，也负责大自然的创造和设计。他完成了不少极有创意的事迹，例如为蝴蝶的翅膀塑形，让每只蝴蝶都有独一无二的印记。我们可以从天王星落入的宫位，看出自己会在哪个生命领域里，产生崭新和独创性的想法及行动。在天王星的领域中，我们不需要遵循普遍和传统的行为模式。

乌拉诺斯和大地之母盖娅(Gaea)结为连理，每到夜晚，天空覆盖大地的时候，他们就不停地孕有小孩。这对多产的夫妻产下了巨人族的泰坦(Titan)、几只独眼巨人(Cyclopses),还有一堆长有一百只手臂和五十颗头的怪兽。乌拉诺斯对自己小孩的外表感到憎恶，于是就不让他们活下来。每当小孩要诞生时，他就把小孩塞回盖娅的子宫，也就是大地的最深处。

这意味着在天王星的宫位里，我们可能会酝酿一些自以为很好的想法，但是将想法付诸行动，具体化地呈现之后，结果却可能不太好。当我们将一些看似十分吸引人的事物落实之后，实际的结果可能会让人失望。我们就像天神乌拉诺斯一样，有时必须埋葬自己的原创想法，然后再试一次。

盖娅的子宫中塞满了这些被赶走的小孩，她当然不太开心，所以她在胸内生产了一些钢，然后打造出一把巨大的镶刀，她恳求小孩用这把镶刀阉割父亲乌拉诺斯。她的小儿子克罗诺斯（土星）非常有责任感，自愿去执行这项任务。乌拉诺斯被阉割的阴茎喷出鲜血，射入盖娅的子宫内，于是就产下了“复仇三女神”(Furies),而当乌拉诺斯的阴茎被扔入大海后，与大海的泡沫融合时，阿佛洛狄忒（金星）就从中诞生了。

天王星会为其落入的领域带来复杂的影响。我们较为原始或土星的一面，也就是克制、谨慎、保守、尊重传统和惧怕未知的倾向，都会“扼杀”天王星的创作冲动。当一个宫位内的天王星遭到抑制时，就会产生“复仇三女神”，她们的名字可以翻译成“忌妒的愤怒”、“报复”以及“永无止境”。

在天王星的领域里，如果太过于依附老旧陈腐的“脚本”或行为模式，那么复仇三女神就会紧追不舍。当对该生命领域的状态感到愤怒时，常会责怪是他人带来了不幸，然后就会在心灵产生有害的、痛苦的遗毒。我们需要极大的能量，才能抑制自己不去做出必要的改变，最后可能会因此而感到筋疲力尽、生病或是孤独。

在天王星落入的宫位中，我们应该勇敢地采取新的行动，并且试图去找出其他刺激或独立的生存之道。即使如此，我们仍然会引来复仇三女神，不过这一次却是由别人来帮自己解开束缚，因为我们的行为也会让这些人感受到挑战、侵犯和威胁。无论是从个人或集体的层面分析，在天王星的宫位里，我们必须特立独行、尝试新的趋势或思潮，同时以进步和革命之名，冒险地瓦解自己和周遭的一切事物。

庆幸的是，阿佛洛狄忒也会从痛苦中现身。她的存在意味着当我们尊重土星的限制和传统、并依此运作时，也可以尝试用一些最和谐、最具创意的方式(金星)创造新生命。我们有时并不会全盘推翻旧有的制度，但仍然可以努力地挪出一点空间给新的想法和兴趣，借此呈现出一些改变。这就是天王星赋予其落入宫位的考验。

天王星在 1781年被发现，那是一个全球意识处于动荡的年代。美国革命刚发生，法国革命即将开始，工业大革命也即将来临，科学时代正要开始。在地球上，个人的尊严以及每个人去质疑权威的权利开始进入了集体意识。

超越土星的限制的第一步不是通常意义下的“灵性”，而是不再随大流，超越机械主义。人格在被超越之前必须被完美化，我们必须将自己内在所是的样子呈现出来，不管我们所受到的是喝彩还是中伤。我们必须接受成为一个改革者、一个天才的孤独负担，我们必须走出自己的部落。

我们可能会不停地抱怨自己的工作、婚姻、无聊的生活，但是如果我们脱掉这些支持，完全裸露地站立在这个令人畏惧和残忍的宇宙面前，还能够做出有意识的、前后一致的生命选择，这才是最高的勇气。很少有什么事比这更让人害怕，这种灵性转化就是天王星纵身一跳的本质。

如果我们不想跳呢？那也是我们的选择之一。就像所有的外行星一样，如果我们不愿意努力，这颗行星就不会对我们的性格贡献什么。但是，它也不会因此而蒸发不见了。对天王星的一种软弱的反应可能是进行没有意义的、象征性的抗争，所有那些天王星的爆发力都被安全地投到一些怪癖里去了。

天王星在星盘上所编织的线索里有一个战斗计划，如果我们不选择战斗的话，那么还有一个非常恐怖的投降协议。如果我们没有通过天王星的考验，我们会感觉自己像是生活在别人的电影里一样，而我们扮演的角色是一个无足轻重的、滑稽的牵线木偶。

```
一个天王星在事业部分的人想成为一个挖蚌壳的人。但是他是一个律师，出生于律师家庭，在“要使劲爬上顶端”的理念熏陶下长大。虽然他很想响应天王星的召唤，将他的法律学位扔出窗外，去买一把挖蚌壳的靶子，但是他始终无法鼓起足够的勇气去这样做。
```

天王星这颗行星与真相、正义、解放、结盟以及平等的理想有关，也与任何挑战现状的集体进步潮流相关。天王星要我们超越过去、成长的背景和生物的限制，甚至超越命运的局限，就像是出身贫困家庭的人，不一定就得当农民。天王星最纯粹的理想是，许多个人结合成为团体，每个人都可以表达自我的独特性，但仍然支持自己隶属的大团体。

天王星也倾向于带来某些扭曲。在天王星所在的宫位有关的生命领域里，我们需要自由和真相，同时又非常害怕被自己创造的事物所捆绑，因此而受限。如果我们总是为了改变而改变，就没有任何事物能在这个领域中落地生根了。如果我们总是在疯狂古怪和天才之间摇摆不定，就会非常想要特立独行，目的只不过是造成一些崩裂，或是引起别人的注意。

天王星落入的宫位也是一个人对“人性”限制最不屑一顾的生命领域。只是一味地假设我们可以超越体能的极限，或是超越天性的本能反应，就如同犯下了“傲慢”的罪行，终会招致惩罚。如同灵机一动的弗兰肯斯坦博士制造出科学怪人，我们也会用同样的手法以进步和发展之名，解开世人的恐惧。当乌托邦理想家无法考量人性的真实面时，最后就会将自己颠覆，反过来对付自己，有时甚至会在过程里连累扼杀了所有的人。

#### 水瓶座↑+↓狮子座

上升点落入水瓶座的人必须退后一步，用客观和超越自我的视角看待人生，才能够将自己的独特性看得更清楚。他们必须用一种抽离的观点来看待事件、人和环境，甚至是自己，才能完整且全面性地了解世事的安排。他们对于“集体”有更高层面的觉知，同时能强烈地意识到自己身处的社会背景，而这些都凌驾于纯主观或个人性的参考体系。

水瓶座是由土星和天王星主宰的，更能为了支持或改善集体的运作而付诸行动。土星面向可能是寻找一个团体，借此加强自我的认同域重要性，而接下来他们就会在意自己是否属于“正确的”团体，是否正在做“正确的”事情。天王星面向却可能会认为，人生不应该只为了增加个人的权力或保障，个人的需求也必须考虑进来。

普罗米修斯相信人类可以过得比现在更好，于是就从天神那里偷走了火，把火的温暖带给人类。上升点在水瓶座的人在面对人生时，最好也带着一种对于未来的新希望和憧憬，也就是相信透过应用正确的理论或概念，情况就可以获得改善。但就像普罗米修斯因为自己藐视的行径，遭受到掌权的天神处罚，许多现代的普罗米修斯也会因为自己的理论或行为逾越或威胁到现存的权威，最后遭到遗责、虐待，甚至是囚禁。

生命的本质就是汰旧换新，过时或是已无效用的旧结构或模式，都会被新的模式和理想所取代，上升点在水瓶座的人通常都能掌握环境中最新的潮流和想法。就外表而言，上升点水瓶座的特征往往是干净、精致、明亮和开朗的外表，他们周遭的空气中常有一种电力或磁性。

与上升点水瓶座最相关的理想就是平等主义，也就是所有人的兄弟情谊、姐妹情谊、团结、公平、平等和正义。为了这些目的，上升点在水瓶座的人时常会认为现存体制中的遗责或压迫，会导致社会的分裂。上升点水瓶座的梦想有时会太过于理想主义，因而忽略了人性中比较贪婪、占有或贪得无厌的那一面。同样地，上升点在水瓶座的人常会因为无法释放内心深处那些较为自我中心或非理性的情感，而感到痛苦，同时也很难接受他人来挑战自己崇高的乌托邦理想。

下降点的狮子座追求个人权力和肯定的欲望，会在水瓶座的冷静、客观或平等主义的理想之下蠢蠢欲动。上升点在水瓶座的人十分热爱平等和自由，却时常会被地位崇高的有力人士所吸引。他们可能会把其他人当成获得权力、地位和重要性的工具。因为他们缺乏一种自我中心的特质，所以要是有其他人让他们在团体中显得很愚蠢，他们很可能会恼羞成怒（这个人可能会被他们冷漠对待好一阵子）。

狮子座通常会热情又强烈地去感受一切，把自己的自我感、荣耀和声望视为最重要的参考体系。为了避免狮子座的反作用力，建议上升点在水瓶座的人无论在内心的想法或是外在的表现上，都不要过于道责狮子座。上升点水瓶座的危险之处就在于对体制运作效率的要求，可能会超越个人的独特性和人道精神，而且还会要求具有创造性的个人特质必须符合体制的标准。更极端的结果就是，水瓶座的梦想可能会变成乔治·奥威尔式的梦魔。

#### 水瓶座

###### 形象

一对平行的波浪线，常常被误解为水，这些线其实是蛇。在伊甸园中，蛇诱惑夏娃吃了智慧树上的果子，上帝将她和亚当逐出了伊甸园，世界有了开端。夏娃所做的举动的意义远不止如此，水瓶座的叛逆行为创造了比安全、智慧更珍贵的特质，他创造了人类的自由。自由是水瓶座的终点，个性、做自己想做的事情、能够选择自己的路的能力，不接受任何人的命令，不管它来自父亲、母亲、总统、牧师或者其他权威人物。

###### 主题

说起来容易做起来难，有巨大的力量压迫个性，来自同伴的压力、符合常规、社会化、想要被别人接受的愿望，如果放任它们掌握自己，我们会被左右和摆布，服务于两个主人：自己的本性，以及周围人的各种妄想，马上就失去自由。水瓶座和符合常规无法融化，就像和平和核弹头无法融和一样。水瓶座的道德敌人就是妥协的诅咒，屈服本能就会失去一切，成为日常生活肥皂剧里的又一个熟悉的角色。

想要征服种族本能，水瓶座必须培养对真相的绝对忠诚，不管后果如何，他必须说出自己所看到的。当自由受到挑战时，不管是通过直接的强迫，还是潜藏的说服，他必须站稳脚跟。水瓶座的目的地是对自我完全真实、绝不屈服的表达，完美的个性化，他必须主动接受放逐者的命运，跟社区的价值和动机永远不一致。

###### 课程

每个城市都有不同的文化，临近城市之间的紧张一直存在。被攻打时，如果水瓶座的城墙在压力下崩溃了，他的文化就会被粉碎。决不能投降，一旦敌人通过了城门，一定是一场灭绝性的大屠杀。胜利的敌人会杀了水瓶的国王，将庙宇中的神像打碎，然后竖立自己的神像。

水瓶只有一个策略，保护那些防御，不让它们受到任何损害，不会有条约，也不会有妥协，只有石头和炮弹的冰冷事实。只是要保护的并不是一个历史社会的文化完整性，而是个人的自由和个性。敌人在城墙外排开，他们挤在城门前，扛着攻城木准备撞门。可以有很多种形式，本质都是文化想要我们服从已经建立的行为模式的压力。

我们内在有一系列的偏好和价值，社会对我们有其它的计划。从会说话开始，我们就被编程，这些程序描述了什么是成功、体面和正常。大多数人进入这些模式是自然的，甚至是有帮助的，但这是水瓶座的死刑。水瓶座必须抵抗，不受文化的胁迫，跟随个性的命令，总是自己做选择，不管这会让周围的人如何愤怒或发出不信的冷笑。“我会是正常的，即使那意味着所有人都认为我是疯狂的。 ”

如果坚持这个策略，社会可能会想要用暴力夺走他的自由，将他扔进监狱或者疯人院。但通常文化的压力是更加微妙的：“如果一直这样做，你永远都不会保住工作。我们会让你挨饿，让你感到不安全和不舒服。如果你继续这样，我们都会嘲笑你，称你为疯子，我们永远都不会认真对待任何你做的事。 ”

水瓶的敌人安插了奸细在城墙里，这些间谍会以爱她的人的形式，穿透水瓶的防御系统，从里面打开城门。他们的确深刻而真诚地爱她，不幸的是，这并不表明他们理解她。他们可能是丈夫或妻子，可能是朋友或父母，当水瓶做选择时，他们对他施加极大的压力，强迫他再次考虑，服从他们的期待。

他们的出发点也许是好的，方式都是奸诈的。这些间谍力图让水瓶相信他有责任背叛自己，让他相信如果他真的爱他们，就不会让他们看着他在这个社会中的与众不同，看着他在一个充满圆洞的世界中做一个方块，并看着他因此而受到高压。面对这些间谍是终极考验，比抵抗那些攻城木难得多。

在维护自由的过程中，水瓶必须坚定如钢铁，以面对吓人和苦涩的考验，他必须准备好伤那些爱自己的人的心。不管他们是多么的失望和痛苦，这些痛苦都来自他们有多么想要将他压进一个他生来不符合的模子。他们的伤痛是真实的，只要一个让步，水瓶就能够减轻它。但是他不能让步，不能假装自己是任何自己所不是的人。

水瓶不是冷淡无情的人，但他常常看起来是这样的。他的路途很严峻，将他带到清澈稀薄的真正个性的平流层。如果他的提升会让那些居住在地上的人失望，他有时候必须学会跟这种痛同处，让这些人心碎是自由必须付出的代价。

###### 优势

“从前每年种玉米之前，我们都会献祭一个处女给雨神，每一年他都会回馈给我们一场雨。但是今年你居然说我们不能献祭任何人，雨会自己到来。 ”一万年前，一个水瓶座听到这些话，但她还是坚持己见。如果她的坚持没有造成她被杀，她的坚持就改变了人类的历史。

今年我们不要献祭一个处女，第一个有这种叛逆思维的女人马上遭受极度的压力。也许村庄里有一千个人，其中有九百九十九个人都认为这是个疯狂的主意。但是水瓶却知道这是真理，这种感觉会支持水瓶，他们知道自己是正确的，即使没有一个人赞同。只是知道真理还无法让一个人抵抗千夫之指，还要有不可改变的、不屈服的固执。一旦他站定了脚跟，南极的风暴也不过是清晨的微风，没有任何东西可以撼动他。

水瓶看到了真相，没有人能改变她的想法。在一件很平常的事情里，思想极端的独立让他看到了别人所无法看到的东西。一般认为天才是极端聪明的人，但智力只是天才的一个工具，没有它也能存在。天才就是以一种没有被教导的方式去思考，以全新方式来看旧问题的能力，水瓶拥有很丰富的天才特质。

###### 劣势

```
拒绝穿除了牛仔服之外的任何衣服，坚持自己在总理面前说脏话的权利，拒绝听除了古典乐之外的任何音乐。
```

要抵抗随顺部落的本能的强大压力，水瓶必须对自己拥有绝对的确认，在内在深处，必须拥有一种无法动摇的确信，不管反对者如何激烈或精彩地否认它，他的认知都是正确的。但这种固执也可能毁掉水瓶，他可能会制造一个关于自己独立性的虚假故事，像死守阵地的士兵一样捍卫它。这些古怪的顽固本身并没有任何害处，但会使水瓶偏离个人化的基本任务。

他不是去捍卫创造自己生活的权利，而是默许了社会的压力，跟随常规的生活轨迹，而不去面对自己该面对的进化难题。将叛逆和自由都倾泻在一个相对安全的生活领域中，天才消失了，不再有叛逆，也没有了革命性的思考。只是在人群中多了一张没有名字的脸，过着可预测的生活，只知道用一些会惹恼别人，但最终并无害处的怪癖，来给自己的生活增添一点色彩。

遵守常规本身并不是罪过，大多数人天生都是这样的人，当我们适应社会时也很舒服。但对水瓶来说就并非如此，常规只是一个面具，当她戴上它时，要付出沉重的代价，过的不再是自己的生活。走在这条黑暗路上的水瓶可能表面上看起来很成功、很稳重、很优雅，富有而风趣，但感觉自己像一个与外界格格不入的局外人，像一个毫无破绽地扮演假身份的外国间谍。

即使那些与她最亲密的人都不理解她，跟他共度关系的起伏时，他们会感觉到他有一种疏离。他显得很远，也许很冷漠或毫无感受，言辞很恰当，也履行责任，对笑话的笑点把握得很好，还能自己说笑话。但是别人不会被蒙蔽，每个人都知道他隐藏了一些最基本的东西。那双清澈而如冰刀般锐利的双目之中空空如也，他的自我不在里面。

### 海王星-灵性

1795年发现者被海王星欺骗了，认为这是一颗恒星，所以就没有特殊对待它。后来，天文学家注意到新发现的天王星轨迹有些异常，有些东西在影响它的运动，他们怀疑这来自一颗远得多的行星的干扰。直到巴黎的一位数学家计算出了它的位置，海王星才最终在1846年被德国人发现。

海王星所代表的体验只能通过自我盔甲上的裂缝来引起我们注意，它们往往通过超自然的干扰到来。我们做梦了，我们有了超感官体验，我们突然有了艺术或者诗歌灵感，我们以为自己有精神分裂的幻觉。

这些海王星的事件间接地指出意识有一个更大的框架，就象天王星的轨道偏移间接表明海王星的存在一样。在我们心理中有一部分从不会跟身份认同，它孑然独立，只是观察。它们指向心灵的一个广大的维度，我们的意识像软木塞一样漂浮在那个维度上。

海王星落在出生星盘的哪个位置，我们就会找到一根暴露在外面的神经，超级敏感和脆弱的地方。幻象、梦、幻觉——这些都是海王的领域，在这里，它拒绝接受“真实世界”的限制。这通常也是一个充满灵感的区域，一个充满愿景的地方，但是在这个领域我们也必须小心不要被没有组织、逃避和幻觉式想法所控制。

波塞冬(Poseidon)象征水，也是掌管海洋、湖泊、河流和地下潜流的天神。他虽然已经在海底占据了一个广大的宫殿，但仍然非常忌妒宙斯在陆地上拥有的广阔领土，十分渴望能够在世上获得更多财宝。波塞冬为了争取阿提卡的主权，与女神雅典娜互斗，结果输了。他又为了争取阿耳戈利斯(Argolis)，向女神赫拉开战，最后仍是失败，他也无法从宙斯的手中夺下埃伊那岛(Acgina)。最后，波塞冬在愤怒和绝望之下，用水淹没了所有无法赢得的土地，又恶毒地让河流干涸。

我们反复无常的情绪就像波塞冬一样，时常渴望那些无法拥有的东西。占星学中的水元素与感情领域有关，在某些方面与波塞冬的作风十分类似。当波塞冬从海中现身时，海水会欢悦地替波塞冬开路，豪气万千地围绕着波塞冬，有时他的出现又会伴随着狂风暴雨。同样地，我们的感觉有时可能会带来满足感，充满着神圣的启发，有时又可能像浪潮般地将我们淹没。

海王星就像月亮和金星一样，也是一种“阴性”能量，代表一种想要消融、适应、反映，同时与他人融合的自我面向。母性的月亮是透过他人的反射，来获得自我的身份认同。迷人的金星则是借由付出来获得回报。灵性的海王星则渴望摆脱自我，与一个更伟大的东西合为一体。孤立的自我（土星）主要的任务就是自我防卫，海王星则渴望能消除孤绝的疆界感，与众生融为一体。

天神撒登（土星，代表了自我建构法则）很怕被天神尼普顿（Neptune，海王星）推翻，担心自己一出生就会被吞噬掉。对于许多人而言，瓦解个人意识是一种很恐怖的想法，因此会将海王星（与众生重新联结的渴望）放逐到无意识的领域里。然而，所有被堆积在地下室的东西，一定有办法钻出地面，出现在家门口的草坪上。被过度压抑的海王星能量是不会就此消失的，反而会伪装成另一副模样来偷袭我们。

在海王星落入的宫位里，可能会不经意地“设置”一种情况，让自己没有其他的选择，只能牺牲个人的追求和渴望，臣服于更大的力量之下。它会让我们无法改变，也无力去缓冲这一切，然后借由这种方式，清除自我意识中至高无上的优越感和隔绝感。经过这一番的净化之后，更高的境界就会散开大门，欢迎我们的到来。

是木星将海王星从土星的暴政中拯救出来的：扩张自我的欲望（木星）最后会打破个人的疏离感，让海王星重获自由。许多人与其害怕自我的瓦解，不如去扩张自我，追寻一种与无限量的存在产生联结的至乐状态，积极地促成这件事的发生。我们可以透过一些方法，建设性地达成这个目标，例如冥想、信仰、祷告、艺术创作，或是对其他人或理想的无私付出，也可能寻求一些危险的管道，例如药物、酒精，或是忘我地投入各种激情中。

如果人们还依稀记得那失落的伊甸园，就会在海王星落入的宫位里，寻找人世间的天堂。我们会认为海王星应该提供一切，并且会对海王星所掌管的领域怀抱着至高的希望，仿佛救赎就在那里似的。我们会在那个宫位里追求一种至乐和至福，当外在世界无法赐予这些美好的事物时，我们当然会感到失望，甚至觉得受伤和怨恨，于是就在这个宫位寻求其他的安慰，通常都是去翻酒柜或是药箱。

但是对某些人而言，由于海王星无法满足欲望所导致的幻灭，反而可以引领自己走向另一个经验领域。不再是只从外在的现实世界中追寻快乐，反而会向内观照。我们最后会发现自己追寻的至乐和至福，其实早已经存在于内心之中，就藏在尼普顿坚定不动的海底城堡里。

我们常会在海王星的宫位里寻找救星，我们会在与海王星所在的宫位相关的生命领域里，粉演受害者或劣势者的角色（同时也可能放弃个人应尽的责任和努力），渴望能有一个人出现，替自己照料一切。相反，有些人则可能扭转这股动力，企图在这个领域里扮演其他人的救星。他们不像土星是出自于“应该”或“必须”的压力去这么做，他们是因为对其他人的痛苦感同身受。有些情形下，我们甚至会在海王星的领域里变成一个流行的形象或是完美的化身，但也可能是丑闻人物，或是人们的替罪羔羊。

只要稍微想象一下海洋之神的面貌，就知道海王星是难以捉摸的，所以在这个生命领域里所追求的事物，也可能会离奇地回避或消失。有很多情况是我们不愿意面对现实，反而会像电影《欲望号街车》的女主角布兰奇·迪布瓦一样，创造出一个美好的幻象，选择性地只看到那些支持自己幻想的部分。我们迟早都会被现实击垮，然后可能重蹈覆撒，或是学到教训：我们永远无法确定在海王星的宫位里会发生什么事。

海王星和以太世界(etheric world)的事物有关，这是一个无法捉摸、度量或是被看见的世界，它是潜藏于形式之下的本质而非形式本身。我们可以透过海王星的宫位一窥更高层或另类的意识状态，或瞥见超越时空的永恒与无限。海王星也与雾、水气和星云有关，在海王星落入的宫位里，我们总是对自己的理想和目标感到困惑又模糊不清，或是习惯性地随波逐流，以为万物如果本为一体，那么发生什么事，对我们而言都没什么差别。

酒神狄俄尼索斯和耶穌都与海王星有关，他们都宣扬要放弃孤立的自我意识，与更具精神性和灵性的整体合而为一。狄俄尼索斯采集了一堆花朵，再利用酒精的辅助，透过感官至乐转换到另一种状态里。这些人对于世俗生活往往不以为意，最后可能会放弃自我，臣服于更大的事物之下。他们一点也不在乎车子停在双黄线上，或是应该回家替丈夫准备晚餐。依此看来，他们已经超越了时间、界线和形式。有些人认为基督耶穌是“海王型大师”，他既是受害者，也是救赎者，他教导人们要“放下”自我，将一切都交给神。

狄俄尼索斯和基督耶酥都曾经历过被肢解的痛苦，他们都增经死去，然后又获得了重生，我们可以在海王星所在的宫位里体验到这种神性。在海王星所在的领域中，我们可能会先经历瓦解，然后才能用另一种面貌重返人间，借此来揭开超越自我境界的真相。心甘情愿、接纳和信仰，都有助于这个过程的进行。在海王星落入的宫位之中，我们有时候会别无选择，一定得这么去做。

海王星要求我们走出自我、渴求和攻击的世界，而不要牺牲我们作为一个个体的正常运作能力。第一部分是相对容易的，第二部分则没有那么容易，在意识褪去自我的外衣时两种行为都会发生。海王星可能作为一个神秘的影响，充满慈悲和灵感；也可能是一个令人软弱的力量，让人容易酗酒和产生幻觉。

#### 双鱼座↑+↓处女座

双鱼座是个复杂又难以理解的星座，所以上升点落入双鱼座可能就会有各种不同的表现方式。就像在代表双鱼座的雕像中，两只鱼是朝着相反的方向游去，这象征着他们心中的冲突和对立，一方面想要建立一个坚定又具体的自我意识，同时又有一股力量鼓励他们消融和超越自我的疆界。上升点双鱼座如果想要解决这个难题，就必须培养一种自我的身份，同时还必须意识到自己是属于一个超越小我的整体。

如果他们的性格过于刻板，可能就无法做到这一点，但如果自我的身份感过于消散，可能又会导致他们无法有效率地过日子。在前面一种情形中，任何自我疆界之外的事物都无法融入到意识之中，而在后面一种情形中，他们又会被自己的敏感和开放性所淹没或覆盖。他们的危险之处在于一方面太过僵化，另一方面又会带来混乱和瓦解。

上升点双鱼座可能会表现在传统上与双鱼座有关的三个层次：受害者、艺术家，或是治疗者（救世主）。受害者通常无法妥善地处理现实的艰困面，于是就会寻求逃避或依靠，也许会把药物和酒精视为解脱的手段。暖昧不明又困惑的上升点双鱼座可能会随波逐流，让其他人来帮他们做决定。有些人可能会试着逃离传统社会体系中日复一日的单调和圈套，转而投入犯罪、地下活动或是犯下其他恶行。

艺术家会被精神上的幻想激发出灵感，然后透过某一种媒介来展现自己的认知和觉察。第三个层次则是治疗者或救世主，这些人会泰献自己的生命去服务他人，努力将一种在扩张的意识领域中才能警见的生命憧憬，落实到现实的人生之中，而且他们会对“外在”世界的痛苦感同身受。程度不同，这三个层次的上升点双鱼座多多少少都会被某种事物接管，借此忘记或是取代较为世俗的形式或是狭隘的人生观。

另一种基调就是栖牲个人意志，在一些极端的情形中，上升点在双鱼座的人可能会一再地创造机会，让其他人来利用他们。他们不仅会经常性地陷在这种众所皆知的困境中，也会欠缺明确的疆界和限制，这意味着他们会沉满在情绪和喜好之中。从长远的角度来看，这些都是一种过度沉溺和自我级灭的模式。

下降点的处女座可以提供他们平衡的分辨力和常识，同时也能教导他们如何健康地去看待谨慎和限制。上升点在双鱼座的人当然可以从他人身上得到这些特质，但是他们最好能在自己身上找到这些特质。有些上升点在双鱼座的人会沉浸在宗教的狂热或是夸浮的使命之中，所以会需要一位伴侣照料他们在日常生活中较为琐碎或是世俗的必需之事。有时候，上升点在双鱼座的人会因为过于理想主义或是太过浪漫，没有意识到自己总是在挑剔或是谴责他人，只因为对方不符合自己的期待。有些上升点在双鱼座的人则会透过拯救或服务他人，发展出自我认同感。

在外表上，双鱼座在第一宫的人脸庞常带有种梦幻、浪漫或神秘的气息。他们可能会有一双水汪汪的大眼睛，给人的整体感觉就是柔软圆润。他们就像黏土一样，可以将自己塑造成各种不同的模样，言行举止也会散发出一种感性和讨好的魅力，足以诱惑他人，甚至迷倒他人。他们有时候会看起来十分无助，勾起他人的同情心而前去拯救他们。

#### 双鱼座

###### 形象

海洋，生命的母亲，海底山峦、发光浮游生物和遗失城市的领域，象征所有无法理解的东西，所有能够被感觉但无法被知道的东西。海洋冲刷着大陆的海岸，在国家之间传递艺术和思想，带来战争、疾病和瘟疫，运输美酒和食物，诗人和音乐家，将各种文化无止境地混合和融合在一起，双鱼的海洋是将所有人连接在一起的，无法说清的神秘流动。

###### 主题

佛教导师建议将世界看成一场梦，人、事、关系、山峦，这些都被看作幻象，只是头脑中无关紧要的一出戏剧。我们常常误解为世界不是真实的，原子和分子都是幻觉，当你刚把脚趾撞到“虚幻”的床脚上时，这可是很难消化的一个观念。重要的不是世界的真实性，更深的是要意识到我们无法直接体验世界，而是体验对世界的意识。意识是头脑中的东西，我们所认为的世界，只是大脑沟壑中发生的生化现象，只是一个图像的幻觉。

这种知识对大部分人没有实际作用，不管是被真狗还是假狗所咬，反正我们都会痛。但双鱼是意识本身的象征，常常跟神秘主义联系在一起，进化路程代表了心智的根本改变。客观的宇宙瞬间消失不见，只剩下庞大的主观反应的网路，完全换了一个档，不是去观察世界，双鱼观察的是心智在观察世界。双鱼的目的地是心智的微调，这个调整不会改变任何东西，但又改变了一切。认识到不管在哪，做什么，看到什么，我们只能遇到一个无法逃避的现实，也就是自己的意识。

###### 课程

```
冷风刮过衣服，寒冷刺骨。锅里的油脂跳起来，在前臂上烧了一个疤。爱人消失了，连续几月，胃里都有一种恶心感。
```
生活中的各种事件总是逼迫我们相信“外在”有一个世界，客观现实独立于心智存在。双鱼必须放开作用与反作用的世界，颠覆客观宇宙的信念，丧失自己的确定性。每天花时间专注在心智上，闭上眼睛，放慢呼吸，停止念头，直接体验意识。不是意识的内容，那些充满头脑的焦虑、理论还有噪音，而是空的，无形的，宁静的意识本身。可以干脆将冥想称为 “隔离”，是一种自然的，有机的功能，不包含神学，也不包括形而上学。通过冥想，心智意识到了自己，跟通过五官一刻不停汹涌而来的信息流暂时断开了连接。

创造性的工作也可以将注意力从客观世界中收回，转向意识本身，让外在世界从中心舞台上退后。一句诗，曲子的音符，油画的图像，这些都来自心智，而不是客观世界。双鱼必须释放创造性灵感，不管是以艺术的形式，还是以秘密白日梦的纯粹幻象形式。在想象的自由玩耍中，将内在世界体验为确实和真实的，不管有多么短暂，赋予主观世界跟外在世界同样的真实度，就像是将视力带给盲人一般宝贵。

我们无法在想象中永久居住，有自己的关系和责任，有身体需要照顾，双鱼也必须生活在现实世界中。但并不需要减慢他们的工作，以愉快和多彩的方式生活，改变看待世界的方式，没有其它东西需要改变。如果双鱼能保持慈悲、助人、没有竞争性的态度，只是看着自己在世俗中的运气潮起潮落，即使是最繁忙和刺激的生活也能够支持他们进化。双鱼并不需要苦行，那是一种太外在的行为。双鱼必须放开世界，认识到意识本身是我们能接触到的唯一实相，也是唯一需要被适应的实相，避免认为物体和事件独立于头脑存在。

###### 优势

如果放手又将何去何从？这个问题会让大部分人绝望地抓住环境和故事，疯狂地努力完成一件不可能的事，在过山车一样的生存场所中，为自己创造一个安全、稳定的生活。但双鱼本能的知道，个性只是巨大意识海洋里漂浮的软木塞，任何时候只要感觉到冲动，他们都能深吸一口气，潜入内在心识之中。那里是双鱼的避难所，可以逃离生活所带来的羞辱和压力。

一个童话世界，充满奇迹和宁静、无限神奇的世界，一个永远都可以进入的世界，总在召唤着双鱼。双鱼比其它星座更能想象，图像总是自发地从深处浮上来，意识中充满了各种幻象和发明。想象不论是带来艺术作品还是白日梦，注意力都被向内收了，远离了本来占据它的外在境遇。

双鱼的人格是变动的，会被弯曲，适应各种变化的情况。理解他人，怜悯他人，同情和善良很容易。双鱼的意识里可以同时装下所有可能的人类观点，只需要想象他们在那个人的状况中，就能毫不费力地在自己流动的意识中，找寻到别人的主观世界。

双鱼拥有本能的对更高层次意识的觉知，从童年起，他们就忙着探索内在的地图，探索心灵的前沿。有些人变得宗教化，另一些迷上了心理学，还有很多对透视能力以及其它的超自然能力有兴趣。不管以哪种方式探索，都代表了双鱼感觉有自我超越的可能性。双鱼可能会以无数不同的方式表达自己，但他们都带着三个核心的洞见：生命是神秘的，表面境遇是它的面纱，意识本身是解开它的关键。

###### 劣势

没有对意识的入迷，双鱼所有的进化工作都会被阻碍，但它必须被管理和自我控制，否则双鱼会变得迷茫不安。他们的头脑中充满各种图像和暗示，情绪席卷他们，如浪潮般恐惧和空洞的梦想，冲刷他们的意识，淹没了人格。如果这些自我深处的爆炸，没有被安全地导入创造和冥想之中，它们将比断头台更致命，也因为缓慢的速度，将比断头太更残忍。双鱼只是坐在那里，眼睛睁得浑圆，任头脑的地下室和地牢中的东西喷涌而出，击碎个性。

双鱼只是随波逐流，为了保持正常人格的外表，大量的努力被付出，没有什么能量去给生命的策略和承诺。工作被接受了，关系被建立了，从这一点开始，双鱼就开始走最小阻力的路。很快地，生命跟随自己的逻辑，像一匹被激怒了的烈马，带着双鱼狂奔，而双鱼只能努力保持不被抛下来。他们会越来越以第三者的角度来看自己的生活，仿佛自己只是一部无法理解的电影中的角色。

空虚感会让敏感的双鱼从客观世界中逃避，进入主观世界中，不是为了获得洞见，也不是因为看到了生命的梦幻本质，只是一种对麻木的寻求。双鱼可能会开始喝酒，发展出对毒品的依赖，可能会将自己沉浸在书本、电视和音乐中，对食物和性产生迷恋，可能一天睡十个小时。

这些行为被用来激发主观意识，以至于客观现实被暂时抹掉了。在这里并没有发生感知和认识过程的转化，进化的问题被丢进背景之中，被浅陋和无意义的替代品所取代。心识通过所有的想象和创造，来逃避这个世界，在每一个阻碍面前掉头而行，最后只能等待命运自己面对一手烂牌。

### 冥王星-史诗

在太阳系的外端，冰冷黑色的地带，环绕着一颗很小的，像瘟疫一样不祥的冥王星。在这个冰冷的边缘地带，太阳是如此遥远，它的光只不过像一颗很亮的星星一般。在冥王星那里，太阳不是那么特殊，地球不过是一颗微尘，我们都是转瞬即逝的微生物。

这颗行星是在1930年被发现的，同时被发现的还有存在主义。面对冥王星就是去面对生命本质的无意义，冥王星将我们带进荒诞的剧院，它让我们面对星云、超新星、以及无尽的时间。它直盯着我们的双眼，宣读最后的审判：“你什么都不是，你的生活是一个玩笑。 ”

我们如何能够吸收这个认知而不毁灭自己呢？要生活在残酷的真相中，那就是我们必须超越人格的戏剧情节。我们必须跟更大的，永恒的东西认同。在逃离荒诞的绝望旅程中，我们只有一条路：我们必须在永恒中留下自己的印记，我们必须改变世界，我们必须将自己的姓氏刻在历史的树上。

如果没有做到这一点，冥王星会强迫我们看到自己存在的终极无意义性。没有任何事情是重要的，我们不过是在一个冰冷的宇宙中等待着灭绝的寄生虫。只是生存是不够的，只是活着什么都不是。我们死去时，一会就被遗忘了。

冥王星代表着另一种“瓦解”的法则，无情地推动着人生的除旧纳新。就像蛇蜕皮一样，仿佛内心深处有股力量在催促着，逼着我们必须脱离老旧又陈腐的人生阶段，进一步地成长和进化。新的事物最终也会变老，也必须退让出场，让下一个阶段接手延续下去。海王星和冥王星这两颗代表地面之下的天神的行星，经常会颠覆性地破坏老旧的结构，强迫我们高举双手，俯首称臣，但是他们的行事风格相当不同。

海王星就像白蚁或蛀虫啃食房屋的根基，慢慢地瓦解坚硬的老旧结构。冥王星却会像突然倒塌的屋顶，千百吨重的砖瓦砸落在头上。冥王星比海王星更加强势，它会不断地增加压力，让压力渐渐累积到顶点，然后再将我们吸灭。海王星会一步步地诱惑我们，让我们相信自己可以透过栖性和痛苦，来获得净化和纯净，冥王星则会确定我们一定得完全消灭老旧的事物，不留一点残余。冥王星确保一个周期循环的结束，或另一个循环的开始，除了改变和死亡，没有任何其他选择的余地。

有一则古老的神话“伊南娜的堕落”（西尔维亚·佩雷拉著作《屈尊为女神》(The Descent to the Goddess)），伊南娜(Inanna)是天上的女神，她总是充满喜悦、活泼又光芒四射。她有一个性格阴郁沉闷的妹妹厄里斯克革(Ereshkigal)，这个名字的意思是“伟大地府的女神”。厄里斯克革住在地府里，代表冥王母性形象。

厄里斯克革的丈夫逝世时，伊南娜决定造访地府，参加妹夫的丧礼。厄里斯克革非但没有优雅地招待伊南娜，反而恶毒又阴郁地凝视着她，还让伊南娜跟其他所有进入地府的人接受同样的待遇。按照一般的程序，有七个通道可以进入地府，每经过一个通道，就必须脱下一件衣物或珠宝。伊南娜礼貌地接受妹妹的要求，穿过所有通道，并且在过程中逐一脱下她的长袍和珠宝，等她抵达地府最深处见到妹妹时早已经一丝不挂，然后她向妹妹“叩首称臣”。

冥王星作风的厄里斯克革要我们卸下那些用来获得自我认同及活力的外在装饰。即使这是一种非常不愉快又严苟的经验，但这则神话告诉我们，我们必须尊重这一股破坏性的力量，并且屈服于它之下，毕竟这一切都是女神的作为，而这也是一种神性，足以代表或展现一种更高系统的中心或权力。我们必须在冥王星落入的宫位中，用这种方式面对厄里斯克革，向她致意。即使她代表黑暗，但她终究是一位女神。

厄里斯克革最后杀了伊南娜，并将她吊在地府的食物挂钩上。这位美丽又高尚的天国女神的身体就这样被弃置而任由腐烂。同样地，在冥王星落入的宫位中，我们也必须去面对心中腐坏的东西。我们会在冥王星所在的领域里，面对自我天性中黑暗又难以辨识的那一面：澎湃激昂的热情和迷恋、对权力的渴望、原始的感官享受、羡慕、忌妒、贪婪、仇恨、愤怒和救赎，以及早期的伤害和痛苦。除非让这些东西浮出台面，将它们转化，适当地重新融入灵魂之中，否则我们很难完整地活着。

这听起来很不愉快，但事实就是如此，普鲁托（即冥王星）也代表埋藏宝藏及隐藏财富的天神。尽管他掀起了飓风骇浪，让我们抛弃了部分的自我，被放逐到无意识的领域里，但就是因为一切都显得如此地遥不可及，我们才会试着有意识地去控制全局。我们可以透过这个过程与失去的能量重新联结，同时也能撷取一些未曾开发的资源和力量。

伊南娜并没有永远困在地府中。据说她曾去过一个危险的地方，为自己安排脱困的退路，以免卡在麻烦中动弹不得。冥王星作风的厄里斯克革可能会让我们自食恶果，但是我们必须将眼光放远，不能任由自己困坐愁城，过着可憎又痛苦的人生。普鲁托会让我们溃不成军，但我们必须像伊南娜一样重返人间，恢复正常的生活——希望能更有自觉、更有智慧、更全面地去面对人生。

伊南娜是在两个名为“哀悼者”的中性矮人的帮忙之下逃离地府的。哀悼者生来矮小又不显眼，他们偷偷地潜入地府，接近饱受苦痛的厄里斯克革。她不仅历经了丧夫之痛，还怀有身孕，当时的她正处于难产的折磨中。旧的事物已经逝去，新的事物同时在诞生。哀悼者并没有为了伊南娜的惨死谴责厄里斯克革，反而与她贴近，怜悯她的处境。他们让她接受“罗杰斯型”(Rogerian)的治疗法，让她尽情地哀嚎和怒骂，回溯自己的痛苦和哀伤。

哀悼者带给人们的教导就是，即使生命发生不幸、黑暗和痛苦，我们仍然要肯定生命的力量。厄里斯克革非常感谢他们接纳了她，承诺会给他们任何想要的报酬，而他们的要求就是让伊南娜复生，后来厄里斯克革也遵守承诺让她复活。最后，伊南娜终于返回地面，而她在经过转化和重生后，也开始用新生命的能量孕有谷物和植物。厄里斯克革作风的普鲁托会毁灭生命，但也会创造新的生命。

与其将痛苦和危机视为耻辱、病变，或是避之唯恐不及的坏事，不将它们视为走向更新和重生过程中的过渡阶段。其次，我们自知无法掌握或超越自己通责、否认或压抑的事物，所以时常会用逃避的态度去处理不愉快的事情。但秘诀却在哀悼者的手中：我们必须注意，并接受厄里斯克革作风的普鲁托，承认这就是人生的一部分，然后让神奇的疗愈自然地发生。

当我们被毁灭，失去了一切曾经珍惜的事物时，当那些曾经被视为自我身份和生命力根源的事物一一被瓦解时，我们仍然可以有些收获。当所有的一切都被剥夺时，我们还有部分的自我是原封不动的，然后就会发现一件事：即使失去了过去执着的自我身份，内心深处仍有一些东西支持着自己。这就是我们在冥王星所在的宫位获得的礼物：部分的自我是无法被毁灭的。冥王星会在一瞬间释放痛苦，我们也会无条件地获得重生，不必再依附外在或相对的物质世界提供的特别“靠山”。

无论冥王星在星图的哪个地方设立祭坛，我们对于那个宫位的事物都不该只看表面，这里的暗号就是复杂和阴谋。在冥王星的领域中，我们必须追寻隐藏的目的，以及潜藏的、无意识里的动机。隔绝的自我当然不喜欢监督自己的毁灭，而普鲁托就是深层核心“自我”的忠实党羽，他会透过冥王星让我们瓦解自我的疆界，释放更多的真我，“超越自我意识的事物就存在于此，我们必须对它屈服。”

冥王星与极端有关。我们可以在冥王星落入的宫位所代表的生命领域里，预期最美好和最槽糕的事情将会发生。当全能的自我受到质疑时，我们很害怕会被级灭，为求自保就会不择手段地控制发生在冥王星所在宫位里的一切，我们甚至会盲目地表现出强迫性或过度执迷的行为。

然而，只要我们能在同一个领域中，承认有一股超越自我的神秘力量存在，并臣服在它之下，就可能发现自我最伟大的力量、最高尚的本质，以及此生灵魂的目的及奉献的目标，并且将这些东西表现出来。我们不仅能在发生的领域中产生巨变，还能将此作为一种触媒或导火线，让其他的领域也产生转变。对于有些人而言，推动历史洪流的力量，会在他们冥王星落入的生命领域里将他们征服，同时发挥作用。

他们拨动琴弦，但我们才是那间让声音回荡和共振的教堂。他们的力量并不是个人的，而是集体的。他们的力量是催化剂而已，是我们自己的力量被反射给了自己。这是一个珍贵的礼物，它必须被找到并且培养，用甜言蜜语将它从躲藏中勾引出来。

冥王星代表了每个人内在的一种特殊智能，智能的特质有赖于冥王星在星盘中的位置。当我们找到它，我们可能会发现自己开始表达一些自己不知道自己有的洞见。当我们以自己的冥王星意识说话时，人们会仔细倾听它，就好像摩西听到燃烧的灌木中上帝的声音一样。影响大众的体验感觉起来如此好，这就是冥王星的恐怖所在。

#### 天蝎座↑+↓金牛座

希腊神话的英雄人物赫拉克勒斯(Hercules)在他的第八次任务中，必须找到一只住在黑暗沼泽洞穴中的九头蛇怪物许德拉(Hydra),并且将它级灭。他一开始试图杀死这只一直待在水中的怪兽，但每次当他砍掉它的一个头时，同一个地方就会再长出三个头。最后，赫拉克勒斯终于想起老师的教诲：“我们必须先跪下才能站起来，必须先投降才能战胜，必须先放弃才能有所收获。”于是他潜入沼泽的底部，紧揪住许德拉的其中一个头，并将许德拉的身体抛到空中。许德拉一离开水面，马上就失去了力量，变得菱靡不振。赫拉克勒斯砍下它所有的头之后，便出现了第十颗由珍贵宝石打造的头，然后赫拉克勒斯就将这个头埋在石头底下。

上升点天蝎座的人无论透过任何形式，都必须去面对黑暗、禁忌、理藏在深处或是破坏性的事物，而且必须使尽全力与这些事物搏斗。有些上升点在天蝎座的人会看到外在的怪兽，与“外边”的黑暗势力及恶魔对抗。对于其他上升点在天蝎座的人而言，许德拉会在他们的灵魂深处诱惑他们，这象征着里面埋藏着破坏性的情绪，例如忌妒、美慕、贪婪、纵欲过度或权力欲过高。弗洛伊德就是太阳落金牛座、上升点落天蝎座，他把许德拉视为“自我”(id),也就是人性中原始、未经琢磨、本能的那一面。

赫拉克勒斯将许德拉高举出水面后，又将它抛到空中，才成功地将它杀死，上升点在天蝎座的人也必须将意识的光芒引进内心黑暗的隐蔽处。如果天蝎座的能量被压抑下来，这股能量就会在内心深处蠢蠢欲动，毒害心灵，最后发出恶臭，破坏人与人之间的气氛。但如果他们用一种冥顽不化的态度，将这股能量完全释放出来，那么其所产生的破坏性后果，也可能会让人无法承受。

与其压抑天蝎座的本性，或是将它完全展现出来，不如学会承认这些感觉的存在，然后将它们转化，改用另一种具有建设性的方式表现出来。就像许德拉被杀死时出现的宝石，负面的情结可能会转化成宝物。很多艺术家都是将自己的热情或愤怒转化成创作性的表达，才成就了许多傲世佳作。伟大的德国作家歌德写出了用恶魔去诱惑浮士德的故事，他的上升点就落在天蝎座。

上升点在天蝎座的人最明显的外表特征，就是一对能穿透人心的深邃双眼，以及洞悉人性的表情。一条蛇在成长的过程中，体内的压力会一天天地增加，当旧皮变得过于紧绷时，蛇就会开始蜕皮。如火山般猛烈的上升点天蝎座也会累积内在的能量，直到全然爆发，但最后必定也会获得重生。无论是出于自愿或是被迫，上升点在天蝎座的人会清除一切旧有的形式和结构，重新建立全新的模式。

上升点落入天蝎座的人天生具有一种深度，强迫他们去追根究底，探索暗藏的意义和动机。他们不会接受任何表面的价值，就像法国民间故事人物蓝胡子的妻子一样，他们一定会打开那扇应该关上的门。上升点天蝎座常会吸引落在下降点的金牛座人进入生命，当天蝎座必须挑战、攻击、破坏和改变时，金牛座就会表现出耐心、稳定、脚踏实地，并且维持现状。金牛座有能力对付上升点天蝎座的猛烈攻击，然后会沉稳地插上一句：“亲爱的，别忘了今天晚餐是八点开始。”上升点在天蝎座的人必须在一段关系中发展出金牛座的特质，借此平衡自己过度混乱又极端强烈的激情。

###### 形象

婚姻中的不和、工作中令人难过的政治，在昨天这些都是装满你头脑的思维，是你世界的中心。今天，你像尸体一样身体绷得紧紧的，一只蝎子一动不动地待在你的肚皮上，只要稍微动一下你就死了，所有这些就像遥远星系中的一颗微弱红星一样不重要了。所有虚假、虚荣、野心都被撕下来，头脑警觉地裸露着，像一块被切割过的宝石一般锋利，随时准备活着或死亡，只有极度强烈的当下存在了，这种态度，这种意识状态，就是天蝎。

###### 主题

星球在宇宙中飞行，周围有百万吨级的石头随机以数百英里每秒的速度呼啸而过，而你在一层薄薄的空气保护层内，跟很多下决心要搞破坏的疯子共处一个空间。有些人有手枪，有些人有核弹，即使逃过了这些也无济于事，你被注定要死的身体所困，脆弱得像狂奔的牛群中间的一朵兰花。

这些想法令人害怕，我们情愿不去看它。忘记自己的出生，避免去想这些无法想象的东西，活在现有的生活里并尽力而为，死亡也只是未来路上的抽象概念。为自己的生活做计划，买保险，头脑集中于两者之间的现在，平静舒服的随波逐流。

但如果有勇气面对死亡周遭的禁忌，我们马上清楚了最重要的是什么。未来消失了，过去蒸发了，全然地集中注意力于此刻，我们前所未有地知道自己是谁、自己想要的是什么，其它所有事情都不再重要。在这个当下，礼貌、虚伪、令人舒服的仪态全部都被撕掉，只有真相残留下来。

将那种诚实、强度和清晰感带入生活中，我们将不再为有过失的过去还有空想的未来而上瘾或负累。蝎子的目标就是好像每一分钟都是生命的最后一分钟那样去生活，将所有的虚伪都烧掉，让无意识变得有意识，不让任何事情藏在恐惧之墙的后面。

###### 课程

如果医生告诉你，你只有六个月可以活了，你有一百万元，可以自由做任何想做的事。有些人可能会疯狂地安排一趟环球邮轮旅行，其他人可能会去弥补过去所做的可耻的事。那些成功适应的人则会安静地坐下来，回归中心，努力感觉他们想要做的是什么。他们不会思考，天蝎基于感受而不是理性而行动，他们直觉地知道逻辑、推理、理性都是错的工具，在这个状况下不合适。

欢乐来自得到自己想要的东西，而我们必须凭感觉来知道自己想要什么，永远无法通过推理来找到它。一直如此，只是死亡让这个原则显得更突出。在生活的十字路口，通常有很多逻辑的选择。逻辑地说，每个人都有很多不同的事业选择，跟很多不同的人结婚的可能性。逻辑太过普遍性，而不个人化，只有一个作用，去除不可能以及荒唐的选择。

要让每一分钟都过得像是最后一分钟，必须让感受跟行动完全调整为一致。天蝎必须摧毁任何阻止感受进入意识的墙，有勇气去感受任何感受，不管这些感受令他如何恐惧，不管它们可能会给生活带来什么影响。思考死亡的震撼会给人带来情感的清晰，能够让天蝎到达这种自我认识的层次。让死亡成为顾问，完全接受人无法逃避死亡的真实性，让恐惧扰动情绪，问出关键问题：“既然我在这里的时间是有限的，那么下一步该做什么？什么对我来说是真正重要的？有哪些承诺和行为模式是基于‘生命永恒’这一疯狂的假设？”

我们必须揭示的感受，就是我们必须快乐和坚决地生活在一个所有人都必然会死的世界上。但感受常常被各种道德或者纵欲的训练所扭曲，如果失去了跟性的联系，压抑会在行动和感受之间创造不和谐，很多情感需要和心理真相就遗落在墙后。

性当中储存有巨大的情绪能量，接受自己在性欲方面的感受，以情绪的、个人的方式来定义性，而不是以肉体的方式来定义它。它更多是一种强烈的情感交互的需要，而不是高潮的需要，优雅而敏感地让性欲成为它本来所是的样子，超越所有诱人的女孩和坏男人想要我们“去做”的欲望，也让所有传教士和假正经的人说“不要做”的声音安静下来。

###### 优势

压抑机制守在意识边界上，将有意识的部分跟无意识部分区分开，他很保守、审慎，将那些可能会令我们不快的，或者让自我形象受损的意识都阻挡在外。天蝎座的压抑机制运作得不太好，猛烈的情绪、破坏性的思维，以及对环境的令人震惊和痛苦的解释，常常在爆发中涌入意识。如果没有这种缺陷，最本质的自我分析工作就不可能完成，那些令人震惊、有时候会改变生活轨迹的感受就无法被意识到。

没有了压抑机制，意识会被情绪所占据，逼它转向内，对自己变得极端诚实。意识开始不停描绘自己的内在地图，完全投入自我认识，寻找任何没被注意到的小细节，要将那块用来隐藏自己不愿想的东西的幕布掀开。没有哪个星座如此无情地内省，如此深刻地洞察自己，天蝎也会用这样锐利的目光扫视周围的世界。

带着本能的怀疑，天蝎在思想层面进入周围人的头脑，想要去理解，找到每个人最深处的动机和最黑暗的秘密。如剑一般锐利和沉默的心识，一定要撕开任何令人舒服的谎言，每个安慰人的假象，以及对生活的虚伪描述。

###### 劣势

太多的自我认识和太少的自我认识，都有各自的致命陷阱，只要屈服于其中任何一个，天蝎所有的渴望、强度和智慧就会变为毒液。太多的自我认识来得太快，让我们无法看清。天蝎自动地，有时盲目地，会不断地挖掘意识的更深处，总是希望找到最终的事实或终极的领悟，借此来转化生活。

意识可能会被那些太复杂的无法被揭露的感受所迷住，有时天蝎发现得太多，以至于他无法面对，跌入郁郁寡欢的低谷。心识无法将自己从深处拉出来，开始焦虑，记录下不可能和无法解决的状况，将这些录音带不停地播放，直到意识的活力变低，进入绝望的筋疲力尽为止。

太少的自我认识也同样危险，尤其当我们故意将一些令自己恐惧或不快的东西阻挡在意识之外时。如果天蝎选择不去想死亡，他对于死亡的本能知识并不会消失，它会在意识之外的潜意识中运作。这种感受还在，继续给行为和态度带来压力，但天蝎却不再能理解这压力的来源在哪里。心识会被焦虑所占据和折磨，总有一种“有什么糟糕的事情就要发生了”的感觉，会将自己跟任何方便的目标相连：我的车就要坏了，老板就要开除我了，我是不是有一个脑瘤等等。

压抑的性也是同样还在，不过我们不知道它的来源在哪。屈服于阴影的天蝎，会发现自己无止境的饥渴，永远无法满足。但心识不知道自己想要什么，无法理解自己的饥渴，它会再次着魔似地、强迫性地追寻这些目标：钱、权力、一尘不染的洁净别墅。走上这条路的天蝎阴郁地躲在黑暗、焦虑、阴险的阴影里，被无法看见的恶魔慢慢吞噬，直到死亡关上自我绝望的帘子。

### 太阳-自我

给万物赋予生命的太阳是太阳系的中心，这里的一切都围绕着它而旋转，它巨大的质量所形成的引力将九个行星都栓在它们的轨道上。在占星上，太阳是人的个性的重心。没有太阳，一个人就会迷失，被一时兴起而相互矛盾的念头所牵制而无法动弹，在空荡的宇宙中失去方向。

太阳如何将将相互有敌意的品质组织在一起，并让它们在同一个人格里和谐共处呢？答案就在我们的核心处，那无法抵抗的、永远精彩而富有吸引力的：自我。自我是精神的聚焦点，就像太阳是太阳系的聚焦点一样。

太阳的任务是有中心，有组织。它是我们每个人内在的各种不同功能的聚焦点，是我们的身份感。它让我们有一种“自己是一个独特的人”的感觉——有自己的感觉、视角和表现生活的方法。太阳也是让我们得以行动的部分，没有它，我们就一片模糊。

自我可能是一种表面上的虚张声势，但是它实际上很胆怯，很犹豫是不是要进入一种形式。太阳必须很小心地被喂养，让它感觉到安全。同时我们必须防止它以一种幻想的辉煌来掩盖自己的不确定性。每个星座都是一种体验的维生素，能为精神带来活力。吞下一粒这样的维生素，你的太阳就更加闪耀。

这个过程是非常微妙的，如果我们做错了，我们最好错在过度地发挥太阳的这个方向，然后生活会慢慢地纠正它。但是如果自我到最后都没有能够以自信和活力的方式来显现，那么所有的一切都失去了。我们就一事无成，不过是一团散乱的心理碎片——缺乏统一性、意志和目的，沦为一只空空如也的箱子。

你是不是认识一个总是被忽略的人？一个不停看电视的人？或者一个行为仪式化，无聊得像自动电梯一样的人？那么他就是一个缺乏太阳能量的人。那些都是相应的症状而已。如果你没有喂养自己的太阳，心灵（ spirit）就死了。

太阳的符号是一个圆圈，圆圈中间有一个点。这个圆圈没有起点，代表了宇宙的浩瀚无界，而中间的点则代表每个人都是独立的个体，也就是必须具备自我身份认同，但这种小我的身份仍属于一个更大的整体。上升点的特质代表了我们发现自己的途径，但太阳星座才是我们最终会发现的自己，或是想要变成的模样。我们可以在太阳所在宫位代表的生命领域中，发展出太阳星座的本质，借此更加了解自己今生的力量、目的和方向。

太阳所在的宫位代表了我们必须在这个领域中凸显自我，也就是发挥影响力，努力地让自己出类拔萃、挺身而出且与众不同。我们会在这个生命领域中将自己与原型的“母亲”区分开来，进一步认清自己的独特性，不再停留在与宇宙众生融合的阶段。就像天上的太阳位于太阳系的中心位置，会对所有围绕它的行星产生影响，出生星图中太阳的位置也代表我们必须在那个领域中培养自我启发的能力，而非只是被动地反应。

就像神话故事中的英雄一样，太阳的宫位则代表了我们会在什么领域中，与命运的猛龙缠斗，勇敢地对抗生命，克服所有阻挡我们前进的障碍和力量。当我们在太阳落入的生命领域中扩张势力、发展自我时，必须得经历痛苦和挣扎。如果成功地通过考验，就可以展现一个更坚定、更协调的“我”。当月亮因为深藏的本能和过去世的行为模式而摇摆不定时，太阳则可以在其所落入的宫位中发挥力量，挑起改变，实践自己的选择，并且创造崭新的可能性。

意识本身无法跟世界互动，它的作用只是去主观地观察和反应。它从来无法绝对地选择一种特定的体验，或者将自己跟其它的隔离开。这部分是自我的作用，它是精神（ psyche）中做选择的部分。而如果没有选择，任何事都无法发生。

但是为了选择，自我必须首先合并。一个带着各种渴望、理想的人格，必须以一种形式来呈现自己。一个神话必须被创造出来，而且对我们来说，它必须是一个具有说服力的神话。这个身份从我们的内在深处升起，并非完全没有基础的荒唐神话，星盘中太阳的状态能帮助我们看到它的根源。

太阳通过将一系列假设塞入潜意识而创造出自我，这些假设我们可能从来都看不到，但是它们一直潜在我们的每次呼吸之下。每个人的生活都反映了他对生活的潜意识假设、自我的结构与太阳，那些塑造自我的假设总是非常随机的，对每个人都不同。

但是对我们个人来说，它们感觉起来并不随意，而总是显得自然和明显，我们无法理解为什么其他人无法看清。太阳在创造自我的同时，也创造了盲目性和敏感性的缺失。我们陷在自我的个性里，而跟生活为我们展现的其它各种选择隔绝。

我们都面对同样的一张白纸：在生与死之间的一片未知。一个人可能会用工作和事业填满它；另一个会去周游世界；第三个成了瘾君子；第四个进了寺庙。那个商人无法理解到底是什么在驱动那个瘾君子；而那个冒险的水手对另一个人会花30年待在一个寺庙的小房间里的选择也困惑得直挠头。

#### 狮子座↑+↓水瓶座

上升点落入狮子座的人会创造一个亟须发展权力、权威和创造力的世界，而这些都是增加自我独特感的必要手段。我们可从太阳所在的宫位，看出上升点落入狮子座的人可以在哪一个特别的生命领域中，最自然地发现自己独一无二的面向，而他们也会在这个领域中寻找爱、钦佩、掌声和影响力。

但有一部分上升点在狮子座的人会把感染力和效力混为一谈。上升点在狮子座的人之所以爱表现自己、炫耀和夸张卖弄，主要是因为他们迫切地想变成大人物，很渴望自己是重要人物。对他们来说，什么都比平凡无奇来得好。上升点在狮子座的人最热衷的事，和大部分的人偶尔会有的渴望一样——理所当然地展现自己的独特之处。

这种人面临的危险就在于，他们的生命态度可能会过度骄傲。有些人会期待其他人把自己视为贵族，但是他们必须付出努力，来赢得想要的尊敬和地位。他们应该学着不要在没有获得自认为应得的荣耀时，马上恼羞成怒转身走开。他们有时候会害怕去证明自己而踌躇不前，以防自己会因为失败而出丑。如果他们没有替个人的权力、欲望和创造力，找到一个健康的释放渠道，有时就会因为这个世界不认同自己的才华，而变得尖酸刻薄又愤世嫉俗。

狮子座与身体的心脏部位相关，所以这个星座可以付出无尽的爱。上升点在狮子座的人就像太阳一样，除了有高贵的姿态，还可以让接受的那一方感受到一种大方、温暖、具有疗愈效果和生命力的能量。但上升点在狮子座的人通常会期待对方有所回报，对他们所付出的一切表示感激和谢意。如果他们能学会付出而不求回报，那就代表灵魂成长已经到达一个新的层次了。

最重要的就是得培养一种正确看待权力、权威和价值的态度。下降点位于水瓶座，所以可能会遇到一些与自己对立的情境或人，迫使他们必须跳脱自己的观点，从别人的角度去看待生命。他们可能还是会受到尊重，但必须考虑大环境的需求，毕竟自己也是其中的一分子。水瓶座落入第七宫代表必须透过分享和促进共同目标，来练习一对一的关系，借此调和狮子座落入第一宫的过度自我中心或自负的心态。

他们如果想要延续一段关系，不让彼此任何一方的性格受到压抑，就必须透过风象水瓶座的客观和公平，让火象狮子座的热情平息下来，并且有所保留（而非完全地释放殆尽）。就外在的特征而言，狮子座通常会表现出一种权势和骄傲的气度，来赢得他人注意，但我们同时得考虑太阳的相位，有些人可能会有“阳光般”的脸蛋和闪闪发亮的双眼。

```
世界前重量级拳王穆罕默德·阿里的上升点就在狮子座，而狮子座的主宰行星太阳落入第六宫。他透过工作领域中的熟练技巧（第六宫），实践崇高感和自我价值（狮子座在第一宫）。心理学家安妮·迪克森(Anne Dickson)是一位肯定自我技巧的专业老师，她的著作《女性自决》(A Woman in Your Own Right),书名就充分展现了火星与上升点合相在狮子座，而主宰行星太阳又落入第十宫。
```

```
乐队收拾好音响离开麦迪逊花园，下一站来到中国的偏远地区进行慈善演出，在那里西方音乐从未被听到过。主唱阔步走向麦克风，吉他手以一个跪地滑的姿势入场，烟雾弹将粉红色的烟吹向场下一万中国人。他们将手握在一起，放在膝盖上，非常困惑，这些美国来的疯子是谁？他们为什么这么激动？乐队成员慢慢意识到了自己所在的处境，纽约观众的热情回应在这里是无法显现出来的，突然之间，他们重新成了那六个来自爱荷华州的年轻人，离家如此的远。
```

###### 主题

巨蟹拉开面纱，心智深处的光彩夺目和变幻显露出来，内在之眼中的海市蜃楼令意识着迷，注意力从外在活动中移开，消融在被动、观察、感受之中，向内部收缩而很少向外投射。通过与意识根部的连接，生命受精了，变得丰富，开始寻求更加明显的表达。炽热的岩浆在巨蟹的外壳底下流动，到了狮子这里，它像宏伟的火山一样喷涌而出，有时也拥有一样的精细度。心智中的内容从这里得到显露，没有东西被锁在想象的地牢里了，每样东西都被赋予形式并表达出来。

狮子必须在世界上留下自己内在过程的实在证据，狮子的工作就是编织一条透明的线索，表达心智，为内在状态创造出外在象征，在解开线索的过程中，人们被引向跟自己最亲密的谜之中。表达是理解狮子的关键点，巨蟹需要秘密，而狮子讨厌秘密。狮子的目的地是发展出个性，这种行为模式完全符合自己的形态，是一种自然的、不加掩饰的、大方的风度，就像芭蕾舞女演员的舞裙紧贴她的身体一般。

###### 课程

狮子必须从头脑中去除所有的小气、挑剔和狭隘，学会对生命说是，恐惧和怀疑在他的辞典中找不到位置，他的路途是一条肯定、正面的路，任何阻碍他进行自我表达的东西都不能被容忍。所有内在的东西都应该表现在外，将自己思想或价值的印记留在世界中，我们就表达了自己的创造驱动。当狮子抵达了终点，就不会在意别人的评价。如果有人盯着他看，他必须用更大的声音为世界表演。人们大笑也没有关系，狮子还是要肯定自己所是的样子。

对生命毫无疑问的信任是狮子的圣杯，当创造渠道更加直接时，狮子会更富有创造力和魅力。诗歌、绘画、装饰等传统创造手段是狮子归于中心的有价值的工具，这些技能为内在生命的表达提供渠道。跳舞、唱歌、讲故事、戏剧是狮子天生的地盘，这些方面的兴趣更值得狮子去发展。在这里，表演者跟观众之间的距离很近，反馈是即时的，能量出去后，马上就会达到目的地。想象显化的方式不仅限艺术，影响一个公司的形式、一个事件或另外一个人都是自我表达。

狮子代表自我的发展而非超越，接受自我的根本荒唐性，然后陶醉在其中，任自我没有任何自我意识地流向世界。狮子的才华必须以更加自然的方式来展露，庆祝自我、创造、不保留任何个人秘密、保持孩子的天真、自然和新鲜，像孩子一样的自我戏剧化、活在此刻的精神就是狮子的本质。玩耍是最简单的进化策略，不通过预先设计，想象冲破“成熟”的面具喷涌而出，显化在生活的舞台中。自我表达是进化的动力，所有创造性、温暖、顽皮和戏剧都是为达到目的而使用的方法。

###### 优势

```
拿破仑，成功的销售办公室和棒球队。一个玩笑、一支钢琴曲调、或是稍加修饰过的真实故事，以一种带着悬念的方式热情讲述。
```
狮子有一颗表演者的灵魂，不管是以什么形式呈现，每个狮子的电路中都存了那么一两个表演节目。只要给出最轻微的鼓励，就会得到很多出自善意的自我表现，鼓鼓掌，舞台的幕布就升起来了。狮子闪耀和大度的个性给他带来天生的领导才能，常常能够在一群毫无组织的人群中，将那些混乱的方向不一致的意见统合起来。

狮子知道如何变得快乐，他的注意力就在此刻，昨日已逝，明天也许永远不会到来。但这一刻狮子就是国王，他知道必须抓住这个变换的瞬间，从中挤出其中所包含的每一个快乐原子。昨天也许是一系列的泡沫和跌倒，明天那些政治家也许真的会开始导弹攻击，但狮子清楚自己此刻是活着的，今天我的身体还在运作，我的兴致很高，我口袋里还有二十块，一起跳舞吧。

###### 劣势

通过表演、创造、表达流出的能量，必须有一个相应的活力的回流，否则狮子会很快筋疲力尽的耗干枯萎。可能是一句“我爱你，很高兴看到你的活力”或是一万人的掌声，不管以什么形式到来，狮子都需要喝彩。狮子的温暖和风度值得称赞，但无意之间，狮子会投射出一个很自信和骄傲的形象。宣扬自己高人一等可不会获得喝彩，人们会贬抑狮子伟岸的自我形象，可能忽略他或尝试让他丧气。

但这只会逼着狮子更加想要表现自己，当他说 “请爱我”，别人听到的却是“我是最伟大的”。狮子需要爱，但骄傲阻止他去要求爱，越通过打动别人来得到安全感，越会让别人反感。恶性循环就开始于这样的误解，极端状态下，狮子会开始表现出所责难的自大、傲慢。狮子必须学习分享他熟知的脆弱感，去要求支持，如果做不到这一点，就会像一个在装满石像的房间里表演的喜剧演员。

### 月亮-感情

月亮，神秘的女王，永远在改变之中，总在经历阴晴圆缺。我们根本不知道现在是什么月相，也不知道能够在哪里、在什么时间看到它。月亮总让我们惊奇，我们无意之间抬头一看，发现它就在那，在明亮的蓝天中如鬼魅般淡淡悬挂。或者它就隐藏起来，哪里都找不到。

太阳在它的环境中凸显自己，月亮则不是这样。月亮似乎总对外在的什么东西做出反应，并将那个反应显示在自己所呈现的样子中。月亮象征了会反应和回应的意识部分、对环境很敏感的部分、跟环境融合的部分，而不是去改造和抵抗环境的部分。

太阳为了实现自己更动感有力的功能，必须过滤掉很多无意识的东西，它必须保卫自我。但月亮可不是这样，它会调整和反应，总是在迁就中。它从无意识之中喷涌而出，揭露出心灵的丰富，不会过滤任何东西。

跟太阳比，月亮要易变得多，它会滋养想象力，在意识中装满各种想象和幻想。月亮代表了感受，纯粹的情感；代表了精神的心情——生活感觉起来如何；代表了易感性——精神底层的非理智性和情绪。月亮的关键词是心，它将“头”的理智留给其它行星功能。

```
喝了一半的波尔多酒瓶反射着炉火的光，被丢在一边。在石头壁炉边，沙发上，你跟自己的新情人坐在一起，谈话中止了。尴尬中，期待中，目光碰触，转开，又碰触。就在这时，月亮从云中探头，将月光从窗口撒入。它的美让你们俩都沉醉，并在你们的眼中填满了黄色的光，你们向彼此靠近。
```

>**来源：**  
>[内在的天空](读书/神秘/内在的天空.md?id=内在的天空)

天上的月亮本身并没有光，只是反射了太阳的光芒。月亮不同于太阳，太阳代表的是我们应该在哪个生命领域付出努力，变成一个有意识的个体，月亮则代表一个人必须在月亮落入的生命领域中，具备一种融入现实、或是因应现实而改变的天性。月亮落入的宫位代表我们在这个生命领域里，可以敏锐地感受到他人的需求和影响，也最容易受到外界的影响。在这个领域中，一个人会比较容易受到习惯或是过去的训练所影响，也可能被家庭或文化的观念、期望、价值和标准所束缚。我们要知道，有些天生的模式可能颇具价值和建设性，但有些却会阻碍或延误新的发展。

当一个人在努力凸显自我、提升自我意识的过程中，偶尔需要休息一下或停下脚步的时候，就会退缩到月亮所在的生命领域。基于每一个人对归属感、安慰或安全感的需求，我们通常会被月亮落入的宫位所代表的生命领域吸引，因为这是我们可以找到或扮演母亲角色的地方。我们会在这里寻求安全感、包容或依靠，同样地，也会在此滋养或支持他人。在月亮的宫位代表的生命领域里，女性通常扮演重要的角色，因为月亮代表基本的女性或阴性(anima)法则。

天上的月亮会经历不同的阶段和循环，有时候是圆满地开展，有时则是隐秘又遮蔽，因此月亮落入的宫位也代表我们很可能在此遭遇波动和起伏，根据自己不时改变的情绪去“经历每个阶段”，有时会很开放，很容易受伤，有时又会封闭自己，退回到自己的世界里。我们可能会在这个领域中表现出退化、幼稚或是缺少安全感的行为。从比较正面的角度来分析，这是和情绪或本能保持联结的地方，这里也可以提供一些具有帮助的习性或记忆，作为我们生存的支柱。

```
哲学家伯特兰·罗素(Bertrand Russell)的巨蟹座落入第九宫宫头，天秤座月亮则落入第十一宫，他的哲学观和世界观（第九宫）就强烈地同情并支持（月亮）人道主义的理想和自由思潮（第十一宫）。
```

>[人生的十二个面向](读书/神秘/人生的十二个面向.md?id=人生的十二个面向)

做出选择是要将我们后路上的桥全部烧掉，月亮则宁愿生活在想象之中，在那里所有的可能性都能够被保持——只需要不选择它们其中任何一个就能做到这一点。它的作用是去感受，而不是行动或者做出决定。

月亮是生命的灵魂，没有了它，所有的体验都是机械的、没有意义的齿轮转动。冷酷的、线性的、每周一早晨要面对的现实对月亮来说没有任何的吸引力，它可能会懒得像块石头，只能做梦。

月亮没有任何道理，它也不在乎有没有道理。它是一种情绪，一种想要去爱和被爱的需要。是爱，是恐惧，是人性的温暖和堕落。月亮是每个生物内在最短暂易逝和脆弱的部分，它是我们的喜悦，带来了快乐和满足。

绝望和厌倦也是月亮的礼物，它们是硬币的另一面。有时月亮所制造的情感可能对我们来说会太过强烈，我们被淹没了，情绪化，抑郁。如果月亮枯萎了，生命之心就变成了黑暗之心，我们就成了随机宇宙中的一个不被重视的心碎者。

太阳可能建造了金字塔和空间站，却是月亮让我们在看到它们的时候眼中闪出惊叹的光。想要快乐，我们必须让我们的月亮更有生命。就像太阳一样，它必须被喂养，而出生星盘给了我们一个配方。

```
当月亮在自我表达和创造的狮子座，会有一种想要表演和获得喝彩的情感需要。这个格局可能代表玩耍、娱乐、有吸引力的特质，但也可能会事与愿违。当表演所产生的活力和侵扰性让人们失去兴趣，没有了喝彩，月亮的胆怯就会接管。没有根据的不被欣赏、不被爱的感觉可能会严重损害这个人，看似完满的生活也会感觉空虚和没有意义。
```

#### 巨蟹座↑+↓摩羯座

当上升点是巨蟹座时，这个人就必须精微地调整情绪，更清晰地认识自己。有些上升点在巨蟹座（尤其是星图其他部分以水元素为主，或者海王星被强化）的人在感情上非常容易受伤，所以必须学会建立一个坚硬的外壳来保护自己。其他上升点在巨蟹座（出生星图中其他部分有大量的风元素或土元素）的人可能没有意识到在自己冷静镇定的表面下，其实是波涛汹涌。这一类的人会吸引一些经验发生在自己身上，提醒他们必须承认、尊重并释放自已的感觉。

螃蟹除了会随时准备缩回壳内，也经常会采取迂回或侧面的路径前进，这反映了上升点巨蟹座不喜欢正面冲突的本质。螃蟹一直待在水里会不舒服，所以会到陆地上探险，然后又退回到水中。上升点在巨蟹座的人也会展现这种有如跳舞般往前两步、往后退缩一步的行为模式。巨蟹座落入第一宫的人，在内心深处往往能感受自己的生理时钟和节奏，而他们应该学会尊重并“跟随”这种感受。从这个角度看来，他们能够“改变”生命的自然循环。

螃蟹有一种不屈不挠的特质，它的大螯会紧抓着东西不放，绝不轻易松开。上升点在巨蟹座的人会紧紧依附着自己的感觉，无论是喜悦或痛苦，都很难让情绪过去，除非有一种更强烈的感觉超越它。对于风象上升星座的人而言，解释、理由和理性化都是非常合理的，但这些特质只会让上升点在巨蟹座的人对你敬而远之，甚至拒之不理。

巨蟹座也与子宫和胸部有关，胸部提供养分，让新生命得以维持下去，而子宫则是生物成长时的完美环境。上升点落入巨蟹座的人可以透过滋养来增加自我的觉知，无论是滋养一个家庭、支持一个商业计划、甚至只是一个感受很深的动机，上升点在巨蟹座的人都可以孕育它，让它“成长为自身的模样”。

但如果他们拒绝粉演一个积极母亲的角色，就可能转换成另一个模样，也就是找其他人来扮演他们的母亲。如果他们过度地认同母亲，就有可能在家族根源的子宫内待得太久，或者会不停地寻找一个已经丧失或不曾拥有的完美母亲，这些都是上升点在巨蟹座的人通常会遇到的问题。

下降点摩羯座引发的特质可以平衡巨蟹座的极端倾向。上升点在巨蟹座的人时常会受到情绪或感情侵袭而淹没其中，如果想要维持一段关系，他们就必须学会分辨哪些情绪是适当且有用的，哪些情绪是必须经过过滤，并加以控制的。他们可以通过伴侣关系，在自己反复无常的感受波动中建立一种架构（摩羯座），也可以通过帮助他人得到成就，完成自己今生必须满足的需求。他们时常会寻找一位能够提供保障、力量和稳定性的伴侣，但上升点在巨蟹座的人必须向内发掘这些特质，而不是从其他人的身上获得这些特质。

上升点落入巨蟹座的人时常有一张“月亮脸”，圆圆的，很祥和，也很讨人喜欢。他们的身材通常是圆润而丰满的，很容易增加体重或是将水分存留在体内。他们的上半身时常特别突出，和身体的其他部分不成比例。

###### 形象

螃蟹肉汁多味美，是海鸥的食物，作为一小块等待着掠食者的柔弱生物，无法以任何办法来保护自己，他的希望何在？为了生存，巨蟹必须长出一个壳，在自己和自然之间长出一层墙，有了这个盔甲，他跟环境就达成了和平。但这个成功下隐藏着冒险转变的种子，巨蟹要吃，要成熟，不久他的壳就容不下他了，壳必须被褪掉。如果他很狡猾和幸运，就能够活到有机会再长出另外一个壳，这个壳会更大，更适合他成长之后的状态。


###### 主题

对双子来说，真理在外面某个地方，他们在世界里搜索着，用警觉和好奇填充自己。但是搜索没有结果，生活比从前更加神秘了，充满了更多未解决的问题和没有消化的细节。被双子的旋转木马转得有些眩晕，被生活中的混乱所困惑，意识现在向内转，转向它的根。巨蟹向着新方向出发了，他不是一路在宇宙中搜寻，而是顺着意识的卷须向内走，探索体验深处的根基，摸索着心的位置。

坐在去火地岛的旅游车上，会让多数人兴奋得像打了一针麻风剂一样。但巨蟹只需要坐下来，闭上眼睛，十秒钟之后就进入了传说中的中土世界。巨蟹能舒服地生活在头脑里的童话世界中，外在世界比想象所创造出来的鲜活内在世界苍白得多。客观的宇宙不再存在，剩下的只有一系列的反应模式，双子简短文字性的、新闻式的意识，变成了一张由印象和个人反应组成的挂毯。

巨蟹来此是要洞察自己内在的生活，他们非常熟悉深层自我的语言——情绪的语言，对心识的这个主观、反应式的部分着迷，生命过程就是一个长期的心理分析过程，通常是自己进行的。巨大的主观性驱动着巨蟹，世界的基本构造是感受，去感受意识、感受生活的每一个细微差别、褪去用来武装自己的麻木的壳——用这个壳来抵挡这个世界的残酷，巨蟹的目的地是看到生命如地狱般的不和谐，但是依旧违背常识地去爱、信任和接受存在带给自己的一切。


###### 课程

深水自动探测器的功用目的是绘制海面下四英里深处的地形，从沉淀物里采集样品，最后毫发无损地返回母舰。海水拍打着它，在它发出的声音中寻找任何弱点和缺陷。在那个完全危险的环境中，防御是最重要的生存策略。如果没有盔甲，脆弱的感应器在毫秒之间就会被压得粉碎。将自己用一层钢铁完全包裹起来虽然能够让它生存，但生命的意义并非生存。

一层两英尺厚的钢墙可以完美地保护这个潜水器，但是如果在防御层上没有开口，潜水器就无法跟环境互动。必须得有摄像机的窗，内在电脑的电线必须跟外在眼睛和耳朵连接，必须有一个储藏空间用来存放从外界收集来的物质。如果将机器保护得太好，就无法完成主要工作，但是如果未保护得足够好，又会被它想要探索的环境毁灭。

巨蟹的“感应器”是最敏感的，没有哪个星座能够以这样的强度去进行感受，感受就是巨蟹生命的一切。但他的内在进程是如此地精微，而生活却是如此地嘈杂，以致于如果没有隔绝，他的神经系统将会崩溃，这些情感电路板可能会超负荷，甚至烧掉。生命必须以一种控制的方式一点点地收进来，一下子放下所有防御无异于自杀。巨蟹的目标就是让自己跟世界的互动强度最大化，但同时又保护自己非常精细的情感灵敏度，在生存的前提下创造最小的防御。

害羞是其中一个防御，白日梦和沉默是巨蟹的伪装，他只有在最无法避免的时候才跟世界发生互动。他戴着一个无名氏的面具，就像沙漠中的原始部族人，他就在我们眼前，但是我们什么都看不见，只能看到旁边的灌木丛。这种防御可能会变得更加复杂，在社交场合，巨蟹可能学会了投射一个非常可信的，三维的全息人格，甚至可能是一个爱交际的形象。在伪装背后，他像间谍一样，拍着X光照片，偷偷地探测周围人的灵魂。

要么戴一个面具，要么就全然裸露地呈现自己，巨蟹的选择很少。危险的是，他的防卫系统可能会变得太过强大，安全会被放在成长之前的首位。如果巨蟹想要进化，他必须蜕掉自己的壳，不过蜕壳的时机必须被精确计算。没有一个受巨蟹影响很重的人，需要在人潮拥挤的地方，站在演讲台上脱下自己的心理裤子。必须非常小心地选择听众，而时机也必须完美无缺。爱永远是一个有风险的赌博，但他必须敞开，必须信任。因为巨蟹如此脆弱，所以他冒的险也非常大。

###### 优势

想象、主观、感受都聚焦在爱之上，善良和关心是巨蟹不经思考的反应，像母亲一样去照顾对巨蟹来说是自然发生的，是内在唯一比自我保护本能更强大的力量。看着另一个人时感到心的柔软，没有竞争，感觉不到恐惧，没有任何别的感受，只想去帮助、疗愈、滋养的冲动。去支持、保护别人是很特殊的爱，但相对来说还是安全的，不是褪下自己的壳，也不是赤裸地坦露自己，真正的爱远比这困难而危险，它永远无法不通过努力而发生。

```
懦弱的人可能在喝了两瓶啤酒之后，壮了胆子搭讪巨蟹，还不如去拿把教堂钥匙开国库黄金大门，成功机会要更大。但是让一个如成吉思汗般凶猛的人带着一颗破碎的心出现，巨蟹马上为他敞开心门。
```

###### 劣势

巨蟹将与他接触的每个人的敏感和苦难都引出来，鼓励他们去感觉到足够安全地笑和哭，过不了十分钟，就会有人坐在他的身边，将心情尽情倾泻出来，那些处于痛苦中的人，会被巨蟹充满慰籍的子宫所吸引。像母亲一样去爱可能是爱的柔软表达，但也是可能只是隐藏自己的另一个外壳，在这个游戏里，最低赌注就是孤独。我们会得到理解和慈悲，而巨蟹也获得了安全感、稳定感和隐身，但没有人在一段亲密关系中真正因此前进。

巨蟹必须非常警惕地提防被扮演母亲角色所诱惑的危险，注意不要让自己包裹在过多的智慧、宽恕和理解之上，以至于自己激荡的内在生活和需要也不被人看见了。在公车上对陌生人戴上母亲的面具没有问题，但它不能成为一段婚姻或友情的主题，关系的双方都被剥脱了人性，这段关系的自然平等性就断裂了，为了避免面对令人不舒服的改变，关系也可能枯萎成一个仪式。

情感的坦露对巨蟹来说很可怕，同样可怕的还有外在生活的不安定和不安全—改变、冒险、经历，驱动白羊和双子的那种存在主义焰火就像一部恐怖电影。一点体验就能够维持很久，不过也需要一定程度的花样和改变，悲哀的是，巨蟹的天生谨慎可能会阻碍这件事的发生。在自己安全的壳里，巨蟹浪漫的创造性火花可能会潜伏数十年，像金牛一样落入一种让精神麻木的乏味和可预测的模式，一直机械地做一份工作，即使这份工作对于成长的意义已经耗尽了。
